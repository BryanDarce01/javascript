# ARRAYS 
En los arrays usamos el signo "[ ]"
Aunque es posible mezclar los tipos de datos en los arrays (arreglos) se recomienda siempre tener un mismo tipo de dato en cada array

Este es el array edades
```javascript

const edades = [ 1,2,3 ]
console.log(edades)
```

Aquí se imprime en formato de tabla
```javascript

const edades = [ 1,2,3 ]
console.table(edades)
```

Para filtrar elementos

```javascript

const nombres = [ 'pedro', 'juan', 'carlos']
const nombres2 = nombres.filter(function(nombre){
    if (nombre === 'pedro')
        return nombre
})
console.log(nombres2)
```


## Map lo que hace es iterar sobre todos los elementos
Sino le ponemos el return, lo que hace es devolver el resto de valores como Indefinidos
```javascript

const nombres = [ 'pedro', 'juan', 'carlos']
const nombres2 = nombres.map(function(nombre){
    if (nombre === 'pedro'){
        return nombre
    }
    else {
        return nombre
    }
})
console.log(nombres2)
```

# Destructuring de arreglos
## De manera antigua en JavaScript
```javascript

const nombres = [ 'pedro', 'juan', 'carlos']
const juan = nombre[1]
console.log(juan)

```

## De manera morderna en JavaScript
Lo que hace es que se buscan los valores por su índice
Se hace de esta manera para solo extraer la variable que nosotros necesitamos
```javascript

const nombres = [ 'pedro', 'juan', 'carlos']
const [ , , carlos] = nombres
console.log(carlos)

```

## Método includes
Devuelte True o False si el argumento que se está pasando, se encuentra en el array

```javascript

const nombres = [ 'pedro', 'juan', 'carlos']
const pedro = nombres.includes.('pedro')
```


## Método some
Devuelve True o False si existe uno de los elementos en el array


## Método find
Devuelve el primer elemento que cumpla una condición
En este ejemplo va a devolver al número 20
```javascript

const numeros = [ 10, 20, 30, 40 ]
const mayor15 = numeros.find.(numero => numero > 15)
console.log(mayor15)
```

## Método every
Devuelve True o False si todos cumplen la condición
En este ejemplo va a devolver False porque no todos los números son mayores a 15

Caso de uso: 
Si quiero saber si todos los clientes ya pagaron
```javascript

const numeros = [ 10, 20, 30, 40 ]
const mayor15 = numeros.every.(numero => numero > 15)
console.log(mayor15)

```

## Método reduce
### Este método toma 2 valores así que se debe de agregar ( ) a la arror function
Este método hace un acumulado de los valores

Caso de uso:
Para calcular el total a pagar si un usuario agregó varios artículos al carrito

```javascript

const numeros = [ 10, 20, 30, 40 ]
const resultado = numeros.reduce((total, numero) => total + numero, 0)

```
