# The For of Loop

Creamos el objeto restaurante y unimos el starterMenu con el mainMenu en una variable.

```javascript

const restaurant = {
    name: 'Classico Italiano',
    numGuest: '20',
    categorias: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
    starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
    mainMenu: ['Pizza', 'Pasta', 'Risotto']

  }


const menu = [...restauran.starterMenu, ...restauran.mainrMenu ];

for (elemento of menu) console.log(elemento);

/*
Resultado: 

Focaccia
Bruschetta
Garlic Bread
Caprese Salad
Pizza
Pasta
Risotto

*/

```

### Aplicamos la destructuración de objetos al crear las variables 'indice' y 'elemento'

```javascript

for (const [indice, elemento] of menu.entries()) {
    console.log(`${indice + 1}: ${elemento}`);
}

/*
Resultado: 

1: Focaccia
2: Bruschetta
3: Garlic Bread
4: Caprese Salad
5: Pizza
6: Pasta
7: Risotto

*/
```
