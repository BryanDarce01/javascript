# Nullish Coalesing
Coalescencia nula en JavaScript es un operador que se utiliza para devolver el primer valor que no sea `null` ni `undefined`. 

### Se representa con  `??`

A diferencia del operador `||` (OR lógico), que devuelve el primer valor "truthy" (no nulo ni indefinido, pero también considera como falsy valores como `0`, `""` o `false`).

El operador `??` solo verifica si el valor es `null` o `undefined`. 

Si el valor a la izquierda es uno de estos dos, entonces devuelve el valor a la derecha. En cambio, si el valor a la izquierda no es `null` ni `undefined`, devuelve ese valor.

```javascript
const pagina = null ?? 1
console.log(pagina); // En este caso devuelve 1

```

```javascript
const pagina = 10 ?? 1
console.log(pagina); // En este caso devuelve 10

```

Este operador considera valores nulos solo a:

1. `null` y `undefined`
2. El número cero y las cadenas de texto vacías, NO SE CONSIDERAN `NULLISH`
