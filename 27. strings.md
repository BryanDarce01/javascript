Las cadenas de texto al ser valores primitivos no tienen métodos, así que lo que hace JavaScript tras bambalinas es convertir las cadenas de texto a OBJETOS.

1. Aquí vemos como se imprime solo el texto

```javascript

console.log(new String ('bryan')); // 'bryan'

```

2. Esto es lo que hace JavaScript, lo convierte a un objeto

```javascript

console.log(typeof new String ( 'bryan')); // 'object'

```

3. Le aplica los métodos que estamos llamando, en este caso SLICE y después lo vuelve a convertir a tipo STRING.

```javascript
console.log(typeof new String ('bryan').slice(1)); // 'string'

```
# Empezamos (❁´◡`❁)

Una cadena de texto es simplemente una secuencia de caracteres entre comillas.
Puedes usar comillas simples ' ', dobles " ", o backticks ``(estos últimos para template literals).

Las STRINGS no se pueden mutar porque son valores primitivos.

Cada carácter en una cadena tiene una posición (índice), comenzando desde 0.

```javascript

let texto = "Hola";
console.log(texto[0]); // H
console.log(texto[1]); // o

```

Algunos métodos: 

```javascript

console.log(texto.length); // 4 (cantidad de caracteres)
console.log(texto.charAt(3)); // a Otra forma de acceder
console.log(texto.indexOf('H')); // 0
console.log(texto.indexOf('o')); // 1
console.log(texto.indexOf('l')); // 2
console.log(texto.indexOf('a')); // 3
console.log(texto.lastIndexOf('a')) // 3

```
En este último caso, los índices son sensibles a las mayúsculas y minúsculas y sino encuentra el caracter específico, devuelve -1.

```javascript

console.log(texto.indexOf('h')); // -1

```

Aquí en la variable 'nombre' podemos ver como en el último ejemplo al no encontrar el nombre 'bryan' en minúscula, devuelve -1.

```javascript

const nombre = 'Mi nombre es Bryan';
console.log(nombre.indexOf('Bryan')) // 13

```

```javascript

console.log(nombre.indexOf('bryan')) // -1

```

## Método SLICE

Si queremos extrar solo ciertos valores de una cadena de texto usamos el método SLICE.

Aquí se toma el primer índice a partir del cual queremos hacer la extracción y también le podemos poner el final.

```javascript

console.log(nombre.slice(3)); // 'nombre es Bryan'
console.log(nombre.slice(3,12)) // 'nombre es'

```

Con el -1 obtenemos el útlimo caracter sino sabemos la longitud.

```javascript

console.log(nombre.slice(-1))

```
En este ejemplo estamos tomando el primer índice que es 0 y el primero que hace match con el método indexOf.

```javascript

const nombre = 'Mi nombre es Bryan';

console.log(nombre.slice(0, nombre.indexOf(' '))); // Mi
console.log(nombre.slice(0, nombre.indexOf('e'))); //'Mi nombr'

```
Aquí le quitamos que busque el índice 0 y solo encuentre el ultimo espacio, por eso imprime el nombre pero con espacio al inicio.

En el segundo ejemplo no imprime lo mismo.

```javascript

console.log(nombre.slice(nombre.lastIndexOf(' '))); // ' Bryan'

console.log(nombre.slice(nombre.lastIndexOf(' ') + 1)); // 'Bryan'

```
Podemos usar números negativos para acceder a los índices.

```javascript

console.log(nombre.slice(-5)); //'Bryan'

console.log(nombre.slice(1, -5)); // 'i nombre es '

```

## Trim();

Si quieres quitar tanto al inicio como al final

```javascript
const email = ' BryAn@Gmail.CoM \n';

const email_login = email.toLowerCase();
const email_correcto = email_login.trim();
console.log(email_correcto); // 'bryan@gmail.com'

```
#### Lo podemos hacer en una sola línea

```javascript

const email_corregido = email.toLowerCase().trim();

console.log(email_corregido); // 'bryan@gmail.com'

```

## REEMPLAZAR VALORES
### replace();

Vamos a reemplazar el signo de euro por el signo de dólar y la coma por un punto

```javascript

const precioEuro = '399,99€';
const precio_dolar = precioEuro.replace('€', '$').replace(',', '.');
console.log(precio_dolar); // '399.99$'

```

## replaceAll();

Si usamos 'replace', cambia solo la primera apareción que haga match.

Por eso si necesitamos cambiar la misma palabra que aparece varias veces, usamos 'replaceAll'.

```javascript

const anuncion_correcto = anuncio.replace('jóvenes', 'adultos');
console.log(anuncion_correcto);

// 'A todos los adultos, deben dirigirse a la oficina de atención al cliente. Todos los jóvenes.'

const anuncio_final = anuncio.replaceAll('jóvenes', 'adultos');
console.log(anuncio_final); 
// 'A todos los adultos, deben dirigirse a la oficina de atención al cliente. Todos los adultos.'

```
### Con expresiones regulares que dio Copilot

La letra 'g' significa Global y la letra 'i' significa que ignora si está en mayúscula o minúscula al buscar coincidencias.

```javascript

const anuncio_modificado = anuncio.replace(/jóvenes/gi, 'adultos');
console.log(anuncio_modificado);

// 'A todos los adultos, deben dirigirse a la oficina de atención al cliente. Todos los adultos.'

```

## SPLIT();

Se usa para dividir una cadena de texto en un array de subcadenas, utilizando un separador que tú especifiques.

```javascript

const lista = "manzana,banana,pera";
const frutas = lista.split(",");
console.log(frutas); // ["manzana", "banana", "pera"]

```

### Usando destructuración

```javascript

const [primerNombre, Apellido] = 'Pedro Perez'.split(' ');
console.log(primerNombre, Apellido); // [ 'Pedro', 'Perez' ]

```

const nombre = ['Mr.', primerNombre, Apellido.toUpperCase()].join('-------');

## padding

Se suele usar cuando queremos rellenar (o completar) una cadena de texto para que tenga cierta longitud, agregando caracteres al principio o al final.

### padStart(cantidad de elementos a agregar, elemento);

En este ejemplo agregamos el signo "+" 10 veces al inicio.

```javascript

const numero = "5";
const padded = numero.padStart(10, "+");
console.log(padded); // '+++++++++5'

```
### padEnd(cantidad de elementos a agregar, elemento);

```javascript

// const nombreCompleto = 'Pedro Perez';
console.log('Pedro'.padStart(10, '+').padEnd(20, '+'));

// '+++++Pedro++++++++++'

```

 COPILOT

```javascript

function guardarTarjetaCredito(numeroTarjeta) {
  const numeroTarjetaString = numeroTarjeta.toString();
  const ultimosCuatroDigitos = numeroTarjetaString.slice(-4);
  const longitud = numeroTarjetaString.length - 4;
  const asteriscos = '*'.repeat(longitud);
  return asteriscos + ultimosCuatroDigitos;
}

console.log(guardarTarjetaCredito(1234567890123456)); // ************3456
console.log(guardarTarjetaCredito(1234)); // 1234

```
