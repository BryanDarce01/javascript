# Operador Ternario

La estructura es:

1. Condición a validar
2. ? 
3. Si la condición es verdadera, se ejecuta esta parte
4. :
5. Si la condición es falsa, se ejecuta esta parte después de los 2 puntos

También se pueden agregar múltiples operadores

|| - OR
&& - AND
< MENOR QUE
> MAYOR QUE

```javascript
const auth = True
const saldo = 100
const deuda = 7
const tarjeta = True

saldo > deuda || tarjeta? 
    console.log('Puedes pagar') :
    console.log('No puedes pagar')

```
