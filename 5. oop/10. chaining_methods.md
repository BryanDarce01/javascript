# Method Chaining

En JavaScript, el method chaining (encadenamiento de métodos) es una técnica en la que puedes llamar a varios métodos sobre el mismo objeto en una sola línea, porque cada método devuelve el propio objeto `(this)`.

## Ejemplo

```javascript

class Calculator {
  constructor(value = 0) {
    this.value = value;
  }

  add(num) {
    this.value += num;
    return this; // 👈 Retorna el objeto para encadenar
  }

  subtract(num) {
    this.value -= num;
    return this;
  }

  multiply(num) {
    this.value *= num;
    return this;
  }

  divide(num) {
    this.value /= num;
    return this;
  }

  showResult() {
    console.log(this.value);
    return this;
  }
}

const calc = new Calculator();
calc.add(10).subtract(2).multiply(3).divide(4).showResult(); 
// (10 - 2) * 3 / 4 = 6

```

## ¿Por qué funciona?

- Cada método modifica el estado del objeto.

- Al final del método, se retorna this (el objeto actual).

- Así, la siguiente llamada de método se hace sobre el mismo objeto.

## Beneficios

- Código más limpio y legible
→ Se evita escribir varias veces la misma variable.

- Patrón fluido (Fluent Interface)
→ Muy usado en librerías como **jQuery, Mongoose, Sequelize**, etc.

## En resumen:
El **method chaining** es encadenar métodos retornando `this`. 

Hace el código más fluido y legible, y se usa mucho en librerías y frameworks.
