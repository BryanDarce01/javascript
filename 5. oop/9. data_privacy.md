# Data Privacy

En programación orientada a objetos **(POO)**, Data Privacy (privacidad de datos) se refiere a la capacidad de ocultar los detalles internos de un objeto y restringir el acceso directo a sus propiedades internas, de manera que solo se pueda interactuar con él mediante métodos controlados.

Esto se conoce también como **encapsulamiento.**

## 1. Public Fields

Son propiedades que pertenecen a cada instancia del objeto, y son accesibles desde cualquier lugar.

```javascript

class Person {
  // Public field
  name = "Anonymous";

  constructor(age) {
    this.age = age; // también es público
  }
}

const p1 = new Person(25);
console.log(p1.name); // "Anonymous"
console.log(p1.age);  // 25

```

- No se necesita `this.name = ...`  en el constructor; el public field ya lo inicializa.

## 2. Private Fields

Empiezan con `#`. Solo son accesibles dentro de la clase.

```javascript

class Person {
  // Private field
  #ssn;

  constructor(name, ssn) {
    this.name = name;
    this.#ssn = ssn;
  }

  getSSN() {
    return this.#ssn; // ✅ se puede acceder desde dentro
  }
}

const p2 = new Person("Pedro", "123-45-6789");
console.log(p2.name);     // "Pedro"
console.log(p2.getSSN()); // "123-45-6789"
console.log(p2.#ssn);     // ❌ ERROR: no es accesible desde fuera

```

## 3. Public Methods

Son las funciones que se definen normalmente en una clase.

Se acceden desde cualquier instancia.

```javascript

class Person {
  constructor(name) {
    this.name = name;
  }

  // Public method
  greet() {
    console.log(`Hi, I'm ${this.name}`);
  }
}

const p3 = new Person("Maria");
p3.greet(); // Hi, I'm Maria ✅

```

## 4. Private Methods

Empiezan también con `#`. Solo se pueden usar dentro de la clase.

```javascript

class BankAccount {
  #balance = 0;

  // Private method
  #log(message) {
    console.log(`LOG: ${message}`);
  }

  deposit(amount) {
    this.#balance += amount;
    this.#log(`Deposited ${amount}`);
  }

  getBalance() {
    return this.#balance;
  }
}

const acc = new BankAccount();
acc.deposit(100);   // LOG: Deposited 100
console.log(acc.getBalance()); // 100
// acc.#log("test"); ❌ ERROR

```

- Útil para funciones internas que no se quieren exponer.

## 5. Static Fields

Pertenecen a la clase en sí, no a las instancias.

```javascript

class Person {
  static species = "Homo sapiens"; // Static field
}

console.log(Person.species); // "Homo sapiens"
const p4 = new Person();
console.log(p4.species); // ❌ undefined, porque no pertenece al objeto

```

- Se usan para **valores o utilidades comunes a todas las instancias.**

## 6. Static Methods

Igual que los fields, pero son funciones. No necesitan instancias.

```javascript

class MathHelper {
  static add(a, b) {
    return a + b;
  }
}

console.log(MathHelper.add(2, 3)); // 5 ✅
const mh = new MathHelper();
// mh.add(2, 3); ❌ ERROR

```

- Se usan para **funcionalidad de utilidad.**

## 7. Combinando todo

```javascript

class Person {
  // Public field
  name;

  // Private field
  #age;

  // Static field
  static species = "Homo sapiens";

  constructor(name, age) {
    this.name = name;
    this.#age = age;
  }

  // Public method
  introduce() {
    console.log(`Hi, I'm ${this.name}, ${this.#age} years old`);
  }

  // Private method
  #getOlder() {
    this.#age++;
  }

  birthday() {
    this.#getOlder();
  }

  // Static method
  static generalInfo() {
    console.log(`We are ${Person.species}`);
  }
}

const p5 = new Person("Ana", 30);
p5.introduce(); // Hi, I'm Ana, 30 years old
p5.birthday();
p5.introduce(); // Hi, I'm Ana, 31 years old
// p5.#age ❌ Error
// p5.#getOlder() ❌ Error

Person.generalInfo(); // We are Homo sapiens

```

## 📌 Resumen:

| Tipo           | Sintaxis                          | Acceso               | Pertenece a |
| -------------- | --------------------------------- | -------------------- | ----------- |
| Public Field   | `name = "X"`                      | Desde fuera          | Instancia   |
| Private Field  | `#age = 30`                       | Solo dentro de clase | Instancia   |
| Public Method  | `greet() {}`                      | Desde fuera          | Instancia   |
| Private Method | `#log() {}`                       | Solo dentro          | Instancia   |
| Static Field   | `static species = "Homo sapiens"` | Clase                | Clase       |
| Static Method  | `static add() {}`                 | Clase                | Clase       |
