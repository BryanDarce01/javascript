# Getters

En JavaScript, los getters y setters son métodos especiales que se usan para controlar cómo se accede y se asignan valores a las propiedades de un objeto.

## 1. Getter (get)

- Se usa para obtener **(LEER)** el valor de una propiedad.

- Parece una propiedad normal cuando se accede, pero en realidad es una función que se ejecuta.

Ejemplo:

```javascript

class Persona {
  constructor(nombre, edad) {
    this._nombre = nombre; // Usamos _nombre para diferenciar la propiedad interna
    this._edad = edad;
  }

  // Getter
  get nombre() {
    return this._nombre.toUpperCase(); // Podemos transformar el valor al devolverlo
  }
}

const p = new Persona("Bryan", 25);
console.log(p.nombre); // "BRYAN"  ✅ (acceso como propiedad, no como función)

```

Aunque parece que accedemos a `p.nombre`, en realidad se está ejecutando el método `get nombre()`.

## 2. Setter (set)

- Se usa para asignar un valor a una propiedad.

- Permite validar o modificar el valor antes de guardarlo.

Ejemplo:

```javascript

class Persona {
  constructor(nombre, edad) {
    this._nombre = nombre;
    this._edad = edad;
  }

  // Setter
  set edad(nuevaEdad) {
    if (nuevaEdad < 0) {
      console.log("La edad no puede ser negativa 🚫");
    } else {
      this._edad = nuevaEdad;
    }
  }

  get edad() {
    return this._edad;
  }
}

const p = new Persona("Bryan", 25);
p.edad = -5;   // "La edad no puede ser negativa 🚫"
p.edad = 30;   // Se asigna correctamente
console.log(p.edad); // 30

```

Aunque parece que hacemos `p.edad = 30`, en realidad se ejecuta el método `set edad()`.

## Resumen de cómo funcionan

- `get` → se usa al **LEER** la propiedad → ejecuta lógica antes de devolver el valor.`

- `set` → se usa al **ASIGNAR** la propiedad → ejecuta lógica antes de guardarlo.

Internamente, siguen siendo funciones, pero desde fuera se usan como propiedades normales.

## En resumen:

- Getters y Setters = Encapsulación.
- Sirven para proteger datos y controlar cómo se acceden o modifican.
- Son clave en Programación Orientada a Objetos (POO).
