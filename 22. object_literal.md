# The Object Literal

## 1. 
### Podemos meter un objeto a otro copiando su nombre

Antes de ES6 era así: "openingHours: openingHours"

```javascript

const openingHours = {
    thu: {
      open: 12,
      close: 22,
    },
    fri: {
      open: 11,
      close: 23,
    },
    sat: {
      open: 0, // Open 24 hours
      close: 23,
    }
}

const restaurant = {
    name: 'Classico Italiano',
    numGuest: 20,
    categorias: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
    starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
    mainMenu: ['Pizza', 'Pasta', 'Risotto'],
    
    // ES6 Enhanced object literal
    openingHours

  }
  
```

## 3. Computing 

#### Hacer cálculos

```javascript

const diasDeLaSemana = ['Lunes', 'Martes', 'Miércoles', 'Jueves', 'Viernes', 'Sábado', 'Domingo'];

const openingHours = {
    [diasDeLaSemana[0]]: {
      open: 12,
      close: 22,
    },
    [diasDeLaSemana[1]]: {
      open: 11,
      close: 23,
    },
    [diasDeLaSemana[2]]: {
      open: 0, // Open 24 hours
      close: 23,
    }
}

```

### Posible destructuración

```javascript

const diasDeLaSemana = ['Lunes', 'Martes', 'Miércoles', 'Jueves', 'Viernes', 'Sábado', 'Domingo'];

// Usamos un objeto vacío para ir llenando la estructura de openingHours
const openingHours = {};

for (const [index, dia] of diasDeLaSemana.entries()) {
  openingHours[dia] = {
    open: index === 0 ? 12 : index === 1 ? 11 : index === 2 ? 0 : 10, // Aseguramos horarios diferentes para cada día
    close: index === 0 ? 22 : index === 1 ? 23 : index === 2 ? 23 : 20, // Los horarios también son distintos
  };
}

console.log(openingHours);

/*

Resultado esperado:

{
  Lunes: { open: 12, close: 22 },
  Martes: { open: 11, close: 23 },
  Miércoles: { open: 0, close: 23 },
  Jueves: { open: 10, close: 20 },
  Viernes: { open: 10, close: 20 },
  Sábado: { open: 10, close: 20 },
  Domingo: { open: 10, close: 20 }
}

*/
```
