# Function Declaration
En este tipo de funciones puedo declarar las variables primero y después usar la función

```javascript

sumar(5, 5);
sumar(2, 2);

function sumar(a, b){
    console.log(a + b);
}
```

# Function Expression
En este tipo de funciones no puede declarar las variables y después llamar a la función porque el código se ejecuta de manera lineal.

También cambia su sintaxis porque primero se declara en una variable.

```javascript

const sumar = function (a, b){
    console.log(a + b);
    
}
```

# Arrow function
Entra en la categoría de **Function Expressions**.

La diferencia es que se incluye el signo `=>` después de los `( )` y es más corta de escribir.


```javascript

const sumar = function (a, b) =>{
    console.log(a + b);
}
// En las Arrow Functions no es necesario agregar la palabra Return si es solo una línea de codigo, si incluye { }, si se agrega el return
const sumar2 = (numero1 = 0, numero2 = 0) => numero1 + numero2;
```

También se le quitan los `{ }`

```javascript
const sumar = function (a, b) => console.log(a + b);

```
