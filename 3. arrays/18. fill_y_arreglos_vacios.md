# Creando arreglos vacíos

Estás creando un array de longitud n, pero con espacios **vacíos (slots vacíos)**, no elementos definidos.



``` javascript

const arr = new Array(5);

console.log(arr); // [ <5 empty items> ]
console.log(arr.length); // 5

```

## ⚠️ Importante:

Este array tiene longitud, pero no puedes usar map, forEach, etc. directamente sobre él, porque los elementos no están inicializados.

``` javascript

new Array(3).map((x, i) => i); // ❌ No hace nada, devuelve [ <3 empty items> 

```

# Fill

El método `.fill()` rellena los elementos de un array con un valor estático, desde un índice inicial hasta uno final (sin incluir el final).

Modifica el array **original (in-place)**.

## 🔧 Sintaxis:

``` javascript

array.fill(value, start = 0, end = array.length)

```

- `value`: el valor con el que quieres rellenar.

- `start` (opcional): índice desde donde comienza el relleno (por defecto `0`).

- `end` (opcional): índice donde se detiene el relleno (no se incluye, por defecto `array.length`).

## ✅ Ejemplo básico: rellenar todo el array

``` javascript

const arr = [1, 2, 3, 4];

arr.fill(0);

console.log(arr); // [0, 0, 0, 0]

```

## 🎯 Ejemplo con rango específico:

``` javascript

const arr = [1, 2, 3, 4, 5];

arr.fill(9, 1, 4);

console.log(arr); // [1, 9, 9, 9, 5]

```
### 🔍 Explicación:

Desde índice `1` hasta antes del `4`, reemplaza con `9`.

## 🧪 Rellenar un nuevo array:

``` javascript

const vacio = new Array(5).fill(1);

console.log(vacio); // [1, 1, 1, 1, 1]

```

## 📌 Casos útiles:

- Inicializar arrays con valores fijos.

- Resetear valores.

- Preparar placeholders (null, 0, false, etc).

## ⚠️ Cuidado con objetos o arrays dentro de fill

``` javascript

const arr = new Array(3).fill({});

arr[0].nombre = 'Hola';

console.log(arr);

// [ { nombre: 'Hola' }, { nombre: 'Hola' }, { nombre: 'Hola'} ]

// Todos apuntan al **mismo objeto** en memoria.

```

## ✔️ Solución correcta:

``` javascript

const arr = Array.from({ length: 3 }, () => ({}));

```
