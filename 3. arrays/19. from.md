# From

## 🔍 Array.from() en JavaScript

- `Array.from()` crea un nuevo array a partir de:
        
        Un objeto iterable (como una string, un Set, un Map, etc.)

        o un objeto parecido a un array (como arguments, NodeList, etc.)

        También puedes usarlo para generar arrays con lógica personalizada.

## 🔧 Sintaxis:

``` javascript

Array.from(arrayLike, mapFn, thisArg)

```

- `arrayLike`: algo iterable o similar a un array.

- `mapFn` (opcional): función que transforma cada elemento.

- `thisArg` (opcional): valor de this en `mapFn`.

## ✅ Ejemplo 1: convertir string a array

``` javascript

const letras = Array.from("Hola");

console.log(letras); // ['H', 'o', 'l', 'a']

```

## ✅ Ejemplo 2: convertir NodeList a array

``` javascript

const elementos = document.querySelectorAll('div');

const arr = Array.from(elementos);

console.log(Array.isArray(arr)); // true

```

## ✅ Ejemplo 3: crear array con lógica personalizada

``` javascript

const numeros = Array.from({ length: 5 }, (_, i) => i + 1);

console.log(numeros); // [1, 2, 3, 4, 5]

```

### 🔍 Aquí length: 5 crea un array virtual de 5 espacios, y (_, i) => i + 1 genera los valores.

## ✅ Ejemplo 4: mapFn para transformación

``` javascript

const dobles = Array.from([1, 2, 3], x => x * 2);

console.log(dobles); // [2, 4, 6]

```

## 🧠 Diferencias con new Array()

| Método                        | ¿Crea elementos reales? | ¿Acepta función de mapeo? |
| ----------------------------- | ----------------------- | ------------------------- |
| `new Array(n)`                | ❌ (slots vacíos)        | ❌                         |
| `new Array(n).fill(0)`        | ✅ (con valores)         | ❌                         |
| `Array.from({length: n}, fn)` | ✅ (personalizado)       | ✅                         |
