# findLastIndex

***findLastIndex*** es similar a findIndex, pero busca desde el final del array hacia el inicio. Retorna el índice del último elemento que cumple una condición dada.

## 🔧 Sintaxis:

``` javascript

array.findLastIndex(callback(element, index, array))

```

- **callback**: función que recibe cada elemento, su índice y el array completo.

- Devuelve el índice del último elemento que cumpla con la condición.

- Si no encuentra ninguno, retorna -1.

✅ Disponible desde ES2023. Asegúrate de usar un entorno moderno (como navegadores actualizados o Node.js reciente).

## ✅ Ejemplo práctico:

``` javascript

const edades = [15, 22, 17, 30, 18];

const index = edades.findLastIndex(edad => edad >= 18);

console.log(index); // 3 (porque 30 es el último que cumple y está en el índice 3)

```

## ❌ Si no hay coincidencia:

``` javascript

const datos = [1, 2, 3];

const index = datos.findLastIndex(num => num > 10);

console.log(index); // -1

```

| Método          | ¿Qué retorna? | ¿Desde dónde busca? |
| --------------- | ------------- | ------------------- |
| `find`          | Valor         | Desde el inicio     |
| `findIndex`     | Índice        | Desde el inicio     |
| `findLast`      | Valor         | Desde el final      |
| `findLastIndex` | Índice        | Desde el final      |
