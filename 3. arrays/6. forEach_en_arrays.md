# forEach

Este ejemplo es solo usando el método **For**

``` javascript

const movements = [100, 200, -150, 300, -50];

for (movement of movements) {
  if (movement > 0) {
    console.log(`Depositaste: ${movement}`);
  }
  else console.log(`Retiraste: ${Math.abs(movement)}`)}

'Depositaste: 100'
'Depositaste: 200'
'Retiraste: 150'
'Depositaste: 300'
'Retiraste: 50'

  ```

Usando un contador

``` javascript

for (const [index, movement] of movements.entries()) {
  if (movement > 0) {
    console.log(`Movimiento número ${index +1 } -- Depositaste: ${movement}`);
  }
  else console.log(`Movimiento de retiro ${ index +1 } -- Retiraste: ${Math.abs(movement)}`)}

```

  Usando el método **forEach**

``` javascript

const movements = [100, 200, -150, 300, -50];

movements.forEach(function(movement){
    if (movement > 0) {
        console.log(`Depositaste: ${movement}`);
  }
    else console.log(`Retiraste: ${Math.abs(movement)}`
        )
    }
)

'Depositaste: 100'
'Depositaste: 200'
'Retiraste: 150'
'Depositaste: 300'
'Retiraste: 50'

```

Usando un contador con **forEach**

``` javascript

const movements = [100, 200, -150, 300, -50];

movements.forEach(function(movement, index) {
    if (movement > 0) {
        console.log(`Movimiento número ${index +1 } -- Depositaste: ${movement}`);
  }
    else console.log(`Movimiento número ${index +1 } -- Retiraste: ${Math.abs(movement)}`
        )
    }
)

'Movimiento número 1 -- Depositaste: 100'
'Movimiento número 2 -- Depositaste: 200'
'Movimiento número 3 -- Retiraste: 150'
'Movimiento número 4 -- Depositaste: 300'
'Movimiento número 5 -- Retiraste: 50'

```

**NOTA**

No puedes usar break, continue o return (para salir del bucle) dentro de un forEach(), porque no es un bucle como for, for...of, etc. 

Es un método de los arrays que ejecuta una función para cada elemento sin forma de detenerse anticipadamente.

for...of sí permite break, continue y return (si estás dentro de una función), lo cual lo hace más flexible si necesitas interrumpir el ciclo según una condición.

**¿Cuándo usar cuál?**

1. Necesitas detener el ciclo con break/continue.	Usa for...of

2. Solo necesitas recorrer todos los elementos sin interrupción. Puedes usar forEach()
