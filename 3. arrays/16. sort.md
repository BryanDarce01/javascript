# SORT 

El método `.sort()` ordena los elementos de un array in-place ***(modifica el array original)*** y por defecto lo hace como si los elementos fueran strings ( **¡incluso si son números!**).

## 🔧 Sintaxis:

``` javascript

array.sort([compareFunction])

```

- Si no se proporciona `compareFunction`, ordena como texto Unicode.

- Si se proporciona una `compareFunction(a, b)`, se usa para definir el orden.

## ⚠️ Ejemplo por defecto (sin función de comparación):

``` javascript

const numeros = [10, 2, 30, 4];

numeros.sort();

console.log(numeros); 

// [10, 2, 30, 4] → pero realmente muestra [10, 2, 30, 4] como strings → mal ordenado

```

## ✅ Orden correcto de números:

``` javascript

const numeros = [10, 2, 30, 4];

numeros.sort((a, b) => a - b); // ascendente

console.log(numeros); // [2, 4, 10, 30]

```

``` javascript

numeros.sort((a, b) => b - a); // descendente

console.log(numeros); // [30, 10, 4, 2]

```

## ✅ Ejemplo con strings:

``` javascript

const frutas = ['manzana', 'Banana', 'naranja'];

frutas.sort();

console.log(frutas); 
// ['Banana', 'manzana', 'naranja'] (mayúsculas antes que minúsculas)

```

## 📌 Ordenar objetos por propiedad:

``` javascript

const usuarios = [
  { nombre: "Ana", edad: 30 },
  { nombre: "Luis", edad: 25 },
  { nombre: "Pedro", edad: 35 }
];

usuarios.sort((a, b) => a.edad - b.edad); // orden por edad ascendente

console.log(usuarios);
// [{Luis, 25}, {Ana, 30}, {Pedro, 35}]

```

## 🧠 Tip rápido: orden alfabético sin distinguir mayúsculas/minúsculas

``` javascript

const palabras = ['pera', 'Manzana', 'banana'];

palabras.sort((a, b) => a.localeCompare(b, 'es', { sensitivity: 'base' }));

console.log(palabras); // ['banana', 'Manzana', 'pera']

```

## 🆚 Resumen:

| Caso                           | Código                                                                  |
| ------------------------------ | ----------------------------------------------------------------------- |
| Números ascendentes            | `arr.sort((a, b) => a - b)`                                             |
| Números descendentes           | `arr.sort((a, b) => b - a)`                                             |
| Strings (caso normal)          | `arr.sort()`                                                            |
| Strings insensible a mayús     | `arr.sort((a, b) => a.localeCompare(b, 'es', { sensitivity: 'base' }))` |
| Objetos por propiedad numérica | `arr.sort((a, b) => a.prop - b.prop)`                                   |
