# FlatMap

## 🔍 flatMap en JavaScript

El método `.flatMap()` combina dos operaciones comunes:

- `map()` → transforma cada elemento.

- `flat(1)` → aplana un nivel de profundidad.

### 🔧 Es más eficiente que hacer .map().flat() por separado.

## 🔧 Sintaxis:

``` javascript

array.flatMap(callback(element, index, array))

```

- `callback` es la función que transforma cada elemento.

- Siempre aplana solo un nivel.

- Devuelve un nuevo array plano.

## ✅ Ejemplo básico:

``` javascript

const arr = [1, 2, 3];

const resultado = arr.flatMap(x => [x, x * 2]);

console.log(resultado); // [1, 2, 2, 4, 3, 6]

```

## 🔍 ¿Qué pasó?

- `map`: genera `[[1, 2], [2, 4], [3, 6]]`

- `flat`: aplana un nivel → `[1, 2, 2, 4, 3, 6]`

## 🆚 Diferencia con `map`:

``` javascript

const r1 = [1, 2, 3].map(x => [x * 2]);
// [[2], [4], [6]]

const r2 = [1, 2, 3].flatMap(x => [x * 2]);
// [2, 4, 6]

console.log(r1); // [[2], [4], [6]]
console.log(r2); // [2, 4, 6]

```

## 📌 Ejemplo útil con strings:

``` javascript

const frases = ["hola mundo", "buenos días"];

const palabras = frases.flatMap(frase => frase.split(" "));

console.log(palabras); // ['hola', 'mundo', 'buenos', 'días']

```
## 🧠 Resumen:

| Método      | Qué hace                                |
| ----------- | --------------------------------------- |
| `map()`     | Transforma cada elemento                |
| `flat()`    | Aplana arrays anidados                  |
| `flatMap()` | Transforma y aplana **al mismo tiempo** |
