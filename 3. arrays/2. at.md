# AT Method

``` javascript

const arrey = [1, 2, 3, 4, 5];
console.log(arrey[0]); // Output: 1
console.log(arrey.at(0)); // Output: 1

```

## Obtener el último elemento del arreglo

Es más fácil y rápido usar el método **At**

``` javascript

console.log(arrey[arrey.length - 1]); // Output: 5
console.log(arrey.slice(-1)[0]); // Output: 5
console.log(arrey.at(-1)); // Output: 5

```
