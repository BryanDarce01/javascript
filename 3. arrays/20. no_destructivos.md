# 🧼 ¿Qué es un método no destructivo?
Es un método que:

- **NO modifica el array original.**

- Devuelve una nueva **copia modificada** del array.

## 🔁 1. toReversed()
### 📌 Es como reverse(), pero no modifica el original.

``` javascript

const arr = [1, 2, 3];

const nuevo = arr.toReversed();

console.log(nuevo); // [3, 2, 1]
console.log(arr);   // [1, 2, 3] ✅ sin cambios

```

## 🔁 2. toSorted(compareFn)

### 📌 Es como sort(), pero no toca el original.

``` javascript

const nums = [4, 1, 3];

const ordenado = nums.toSorted((a, b) => a - b);

console.log(ordenado); // [1, 3, 4]
console.log(nums);     // [4, 1, 3] ✅ intacto

```

## ✂️ 3. toSpliced(start, deleteCount, ...items)

### 📌 Es como splice(), pero no modifica el original.

#### Se usa para eliminar o reemplazar elementos sin afectar el array base.

``` javascript

const frutas = ['manzana', 'banana', 'pera'];

const nuevo = frutas.toSpliced(1, 1, 'naranja');

console.log(nuevo);   // ['manzana', 'naranja', 'pera']
console.log(frutas);  // ['manzana', 'banana', 'pera'] ✅ sin cambios

```

## 4. with()
El método .with() te permite crear una copia del array con un solo valor reemplazado en una posición específica, sin modificar el array original.

## 🆕 Ejemplo de .with(index, newValue)

``` javascript

const arr = [10, 20, 30];

const nuevo = arr.with(1, 99);

console.log(nuevo); // [10, 99, 30]
console.log(arr);   // [10, 20, 30] ✅ original intacto

```

## 🔧 Sintaxis:

``` javascript

array.with(index, newValue)

```
- `index:` la posición que quieres reemplazar.

- `newValue:` el nuevo valor que quieres poner en ese índice.

- Devuelve una nueva copia del array con el cambio aplicado.

- Si el `index` está fuera del rango, lanza un `RangeError`.


## ✅ Ejemplo básico:

``` javascript

const original = [10, 20, 30];

const nuevo = original.with(1, 99);

console.log(nuevo);    // [10, 99, 30]
console.log(original); // [10, 20, 30] ✅ original intacto

```
## ❌ Si el índice está fuera de rango:

``` javascript

const arr = [1, 2, 3];

arr.with(5, 10); // ❌ Error: RangeError: Invalid index

```

## 🧠 ¿Por qué es útil?
Antes, si querías reemplazar un solo valor sin modificar el array, tenías que hacer una copia manual:

``` javascript

const arr = [10, 20, 30];

// forma "vieja"
const copia = [...arr];
copia[1] = 99;

```

``` javascript

const copia = arr.with(1, 99);

```
## 🧪 Ejemplo con objetos:

``` javascript

const personas = [
  { nombre: "Ana" },
  { nombre: "Luis" },
  { nombre: "Pedro" }
];

const nuevoArray = personas.with(1, { nombre: "Lucía" });

console.log(nuevoArray);
// [
//   { nombre: "Ana" },
//   { nombre: "Lucía" },
//   { nombre: "Pedro" }
// ]

```
## 🧼 Resumen .with()

| Método               | ¿Modifica el original? | ¿Qué hace?                                   |
| -------------------- | ---------------------- | -------------------------------------------- |
| `array[index] = val` | ✅ Sí                   | Reemplaza directamente                       |
| `array.with(i, v)`   | ❌ No                   | Crea una copia con solo un valor reemplazado |

## 🧠 Comparación resumen:

| Método clásico | Destructivo | Equivalente no destructivo | Ejemplo              |
| -------------- | ----------- | -------------------------- | -------------------- |
| `reverse()`    | ✅           | `toReversed()`             | `arr.toReversed()`   |
| `sort()`       | ✅           | `toSorted()`               | `arr.toSorted(fn)`   |
| `splice()`     | ✅           | `toSpliced()`              | `arr.toSpliced(...)` |


## 🔧 Compatibilidad

- Están disponibles en ES2023.

- Ya funcionan en navegadores modernos y Node.js ≥ 20.
