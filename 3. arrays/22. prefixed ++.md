# Prefidex ++

Cuando usas ++x, el valor de x se incrementa inmediatamente, y el nuevo valor es el que se devuelve en la expresión.

## 📌 Ejemplo:

``` javascript

let x = 5;
let y = ++x;

console.log(x); // 6
console.log(y); // 6  ← porque se incrementó antes de asignar

```

## 🔄 Diferencia con x++ (postfijo)

- `++x`: primero incrementa, luego devuelve.

- `x++`: primero devuelve, luego incrementa.

``` javascript

let x = 5;
let y = x++; // y toma el valor antes del incremento

console.log(x); // 6
console.log(y); // 5

```
## 🧠 En resumen:

| Forma | Qué hace primero | Valor que devuelve |
| ----- | ---------------- | ------------------ |
| `++x` | Incrementa       | Nuevo valor        |
| `x++` | Devuelve         | Valor original     |
