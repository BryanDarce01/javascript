# Object.groupBy

Es una nueva funcionalidad introducida en ES2024 (ES14) y muy útil para agrupar elementos de un array según una condición.

`Object.groupBy()` permite agrupar los elementos de un array en un objeto, según una función de agrupamiento que tú defines.

## 🔧 Sintaxis:

``` javascript

Object.groupBy(array, callback)

```

- **array:** el array que quieres agrupar.

- **callback:** función que retorna la clave del grupo donde debe ir cada elemento.

## ✅ Ejemplo básico: agrupar por tipo

``` javascript

const datos = [1, 'a', 2, 'b', 3, 'c'];

const agrupado = Object.groupBy(datos, item => typeof item);

console.log(agrupado);
/*
{
  number: [1, 2, 3],
  string: ['a', 'b', 'c']
}
*/

```

## ✅ Ejemplo con objetos: agrupar por propiedad

``` javascript

const personas = [
  { nombre: "Ana", rol: "admin" },
  { nombre: "Luis", rol: "user" },
  { nombre: "Pedro", rol: "admin" },
  { nombre: "Marta", rol: "user" }
];

const porRol = Object.groupBy(personas, persona => persona.rol);

console.log(porRol);
/*
{
  admin: [
    { nombre: "Ana", rol: "admin" },
    { nombre: "Pedro", rol: "admin" }
  ],
  user: [
    { nombre: "Luis", rol: "user" },
    { nombre: "Marta", rol: "user" }
  ]
}
*/

```

## 📌 Nota:

`Object.groupBy` siempre devuelve un objeto ordinario ({}).

Si quieres un `Map`, existe también `Map.groupBy()`, que es similar pero devuelve un `Map`.
