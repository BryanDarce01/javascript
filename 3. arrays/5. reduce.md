# REDUCE

Sive para reducir todos los elementos de un arreglo en uno solo.

Por ejemplo sumar todos los números de un arreglo.

![reduce](/img/reduce.png)

``` javascript

const movement = [100, 200,-300, 400, -500];

const avg = movement.reduce((acc, move) => acc + move, 0)
console.log(avg); // -100

```

``` javascript

const balance = movement.reduce(function (acc, move, index, arr) {
  console.log(`Iteration ${index}: Current balance is ${acc}`);
  return acc + move;
}, 0)

'Iteration 0: Current balance is 0'
'Iteration 1: Current balance is 100'
'Iteration 2: Current balance is 300'
'Iteration 3: Current balance is 0'
'Iteration 4: Current balance is 400'
'Final balance: -100'

```
