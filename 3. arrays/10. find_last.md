# FindLast

***findLast*** en JavaScript es un método de los arrays que sirve para encontrar el último elemento que cumple con una condición específica. 

Es similar a find, pero empieza a buscar desde el final del array hacia el 
inicio.

## 🔧 Sintaxis:

``` javascript

array.findLast(callback(element, index, array))

```

- ***callback*** es una función que se ejecuta en cada elemento (de atrás hacia adelante).

- Devuelve el valor del último elemento que cumple la condición.

- Si ninguno cumple, devuelve undefined.

📌 Nota: **findLast** está disponible desde ES2023, por lo tanto no está soportado en navegadores muy antiguos.

## ✅ Ejemplo:

``` javascript

❌ Si no encuentra coincidencias:
const edades = [15, 22, 17, 30, 18];

const ultimaMayorDeEdad = edades.findLast(edad => edad >= 18);

console.log(ultimaMayorDeEdad); // 30 (porque es el último valor que cumple edad >= 18)

```

## ❌ Si no encuentra coincidencias:

``` javascript

const valores = [1, 3, 5];

const result = valores.findLast(num => num > 10);

console.log(result); // undefined

```
