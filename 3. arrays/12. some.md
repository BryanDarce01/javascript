# Some

El método ***.some()*** se usa para verificar si al menos un elemento del array cumple con una condición. 

Si alguno la cumple, devuelve true; si ninguno, devuelve **false**.

## 🔧 Sintaxis:

``` javascript

array.some(callback(element, index, array))

```

- ***callback*** es una función que se ejecuta para cada elemento.

- Retorna un booleano (true o false).

## ✅ Ejemplo básico:

``` javascript

const numeros = [1, 3, 5, 8];

const hayPares = numeros.some(num => num % 2 === 0);

console.log(hayPares); // true (porque 8 es par)

```

## ❌ Si ningún elemento cumple:

``` javascript

const edades = [12, 14, 15];

const hayAdultos = edades.some(edad => edad >= 18);

console.log(hayAdultos); // false

```

## 🧠 Casos de uso comunes:

- Verificar si hay al menos un elemento válido.

- Validaciones rápidas.

- Chequeos de existencia.
