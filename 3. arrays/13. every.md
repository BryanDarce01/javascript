# Every

El método ***.every()*** verifica si todos los elementos de un array cumplen con una condición.

- Si todos la cumplen → devuelve true.

- Si alguno no la cumple → devuelve false.

## 🔧 Sintaxis:

``` javascript

array.every(callback(element, index, array))

```

- ***callback:*** función que se ejecuta para cada elemento.

- Retorna un booleano.

## ✅ Ejemplo básico:

``` javascript

const edades = [22, 25, 30, 28];

const todosMayoresDe18 = edades.every(edad => edad >= 18);

console.log(todosMayoresDe18); // true

```

## ❌ Si un solo elemento no cumple:

``` javascript

const numeros = [2, 4, 6, 7];

const todosPares = numeros.every(num => num % 2 === 0);

console.log(todosPares); // false (porque 7 no es par)

```

## 🧠 Diferencia con **some:**

| Método  | ¿Qué pregunta? | Resultado si...                   |
| ------- | --------------------------------- | --------------------------------- |
| `some`  | ¿Hay **al menos uno** que cumpla? | `true` si al menos **uno** cumple |
| `every` | ¿Todos cumplen?                   | `true` si **todos** cumplen

## 📌 Ejemplo con objetos:

``` javascript

const usuarios = [
  { nombre: "Ana", activo: true },
  { nombre: "Luis", activo: true },
  { nombre: "Pedro", activo: false }
];

const todosActivos = usuarios.every(user => user.activo);

console.log(todosActivos); // false (Pedro no está activo)

```
