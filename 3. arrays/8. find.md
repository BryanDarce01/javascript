# FIND

Regresa solo el **primer** elemento basado en una condición.

No regresa un arreglo.

## Resumen

| Método          | ¿Qué retorna? | ¿Desde dónde busca? |
| --------------- | ------------- | ------------------- |
| `find`          | Valor         | Desde el inicio     |
| `findIndex`     | Índice        | Desde el inicio     |
| `findLast`      | Valor         | Desde el final      |
| `findLastIndex` | Índice        | Desde el final      |
