# FindIndex

Es un método que se usa con arrays y sirve para encontrar el índice (posición) del primer elemento que cumpla con una condición específica.

``` javascript

array.findIndex(callback(element, index, array))

```

- ***callback*** es una función que recibe:

    - element: el elemento actual del array.

    - index (opcional): el índice del elemento.

    - array (opcional): el array completo.

- Devuelve el índice del primer elemento que cumple con la condición.

- Si ningún elemento cumple, devuelve -1.

## ✅ Ejemplo básico:

``` javascript

const numeros = [10, 20, 30, 40];

const index = numeros.findIndex((num) => num > 25);

console.log(index); // 2 (porque 30 es el primer número > 25 y está en la posición 2)

```

## ❌ Ejemplo si no encuentra nada:

``` javascript

const nombres = ['Ana', 'Luis', 'Pedro'];

const index = nombres.findIndex((nombre) => nombre === 'Carlos');

console.log(index); // -1

```

## Diferencia con find:

- find devuelve el valor que cumple la condición.

- findIndex devuelve el índice.
