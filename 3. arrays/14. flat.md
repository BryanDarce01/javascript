# Flat

El método `.flat()` se usa para aplanar arrays anidados, es decir, convertir arrays dentro de arrays en un solo array más plano.

## 🔧 Sintaxis:

``` javascript

array.flat([depth])

```

- `depth` (opcional): el nivel de profundidad que quieres aplanar.

- Por defecto es 1.

- Si quieres aplanar completamente, puedes usar `Infinity`.

## ✅ Ejemplo básico:

``` javascript

const arr = [1, 2, [3, 4]];

const plano = arr.flat();

console.log(plano); // [1, 2, 3, 4]

```

## 🔁 Ejemplo con varios niveles:

``` javascript

const arr = [1, 2, [3, [4, 5]]];

console.log(arr.flat(1)); // [1, 2, 3, [4, 5]]
console.log(arr.flat(2)); // [1, 2, 3, 4, 5]

```

## 🔁 Aplanar todo sin importar cuántos niveles:

``` javascript

const arr = [1, [2, [3, [4]]]];

console.log(arr.flat(Infinity)); // [1, 2, 3, 4]

```

## ❌ Sin flat:

``` javascript

const arr = [1, [2, [3]]];
console.log(arr); // [1, [2, [3]]]

```
