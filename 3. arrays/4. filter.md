# FILTER

Sirve para filtrar los elementos de un arreglo bajo una condición y regresa un nuevo array


![filter](/img/filter.png)


``` javascript

const movement = [100, 200,-300, 400, -500];

const positiveMovev = movement.filter((move) => move > 0);
const negativeMovev = movement.filter((move) => move < 0);

console.log(positiveMovev); // [100, 200, 400]
console.log(negativeMovev); // [-300, -500]

```
