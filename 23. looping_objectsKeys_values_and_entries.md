# Obteniendo las llaves de un objeto

## Object Keys

Creamos el objeto 'openingHours'

```javascript

const diasDeLaSemana = ['Lunes', 'Martes', 'Miércoles', 'Jueves', 'Viernes', 'Sabado', 'Domingo'];
const openingHours = {
    [diasDeLaSemana[0]]: {
      open: 1,
      close: 1,
    },
    [diasDeLaSemana[1]]: {
      open: 2,
      close: 2,
    },
    [diasDeLaSemana[2]]: {
      open: 3,
      close: 3,
    },

    [diasDeLaSemana[3]]: {
      open: 4,
      close: 4,
    },

    [diasDeLaSemana[4]]: {
      open: 5,
      close: 5,
    },

    [diasDeLaSemana[5]]: {
      open: 6,
      close: 6,
    },

    [diasDeLaSemana[6]]: {
      open: 0,
      close: 0,
    }
}

```

### Guardamoe en una variable sus llaves 

```javascript

const propiedades = Object.keys(openingHours);
console.log(propiedades);

// Resultado: Array(7) [ 'Lunes', 'Martes', 'Miércoles', 'Jueves', 'Viernes', ... ]

```

### Forma 1 de imprimir cada una de las llaves

```javascript
let resultado = `Abrimos el restaurante ${propiedades.length} días a la semana. Días que abrimos: ${propiedades.join(', ')}.`;
console.log(resultado); 

// Resultado: 'Abrimos el restaurante 7 días a la semana. Días que abrimos: Lunes, Martes, Miércoles, Jueves, Viernes, Sabado, Domingo.'

```

```javascript


let resultado1 = `Abrimos el restaurante ${propiedades.length} días a la semana. Días que abrimos: `;
for (const dia of propiedades) {
    resultado1 += `${dia},`;
}
console.log(resultado1); 

// Resultado: 'Abrimos el restaurante 7 días a la semana. Días que abrimos: Lunes,Martes,Miércoles,Jueves,Viernes,Sabado,Domingo,'

```

# Object Values

```javascript

const valores = Object.values(openingHours);
console.log(valores);

// Resultado: Array(7) [ { open: 1, close: 1 }, { open: 2, close: 2 }, { open: 3, close: 3 }, { open: 4, close: 4 }, ... ]

```

# Object Entries

Este código imprime las llaves y los valores juntos en un array.

```javascript

console.log(entradas);
```

### Ahora vamos a aplicar la destructuración de un array

Vamos a acceder a las llaves y a sus valores usando '{}'

#### Opción por el chat

```javascript

const entradas = Object.entries(openingHours);

for (const [dia, horas] of entradas) {
    console.log(`El restaurante abre el ${dia} a las ${horas.open} y cierra a las ${horas.close}`);
}

/*

Resultado:

'El restaurante abre el Lunes a las 1 y cierra a las 1'
'El restaurante abre el Martes a las 2 y cierra a las 2'
'El restaurante abre el Miércoles a las 3 y cierra a las 3'
'El restaurante abre el Jueves a las 4 y cierra a las 4'
'El restaurante abre el Viernes a las 5 y cierra a las 5'
'El restaurante abre el Sabado a las 6 y cierra a las 6'
'El restaurante abre el Domingo a las 0 y cierra a las 0'

*/

```

#### Opción por el curso

Aquí aplicamos la destructiración de un objeto usando '[ ]' y '{ }' para acceder a las llaves y valores del objeto.

```javascript

const entradas = Object.entries(openingHours);

for (const [llave, {open, close}] of entradas) {
```
