# MAP

Es un tipo de colección que permite almacenar pares de clave-valor. Es similar a un objeto, pero tiene algunas diferencias importantes que lo hacen más flexible en ciertos escenarios.

1. Claves de cualquier tipo:

    A diferencia de los objetos, las claves de un Map no están limitadas a ser solo cadenas de texto o símbolos. Pueden ser de cualquier tipo, como números, objetos, funciones, etc.

2. Orden de inserción:

    Los elementos en un Map se almacenan en el orden en que fueron insertados. Esto significa que si iteras sobre un Map, los elementos se devolverán en el mismo orden en que se añadieron.

3. Métodos útiles:

    set(key, value): Añade un nuevo par clave-valor al Map.

    get(key): Obtiene el valor asociado a la clave dada.

    has(key): Verifica si una clave existe en el Map.

    delete(key): Elimina el par clave-valor de un Map.

    clear(): Elimina todos los pares de clave-valor de un Map.

    size: Propiedad que devuelve el número de pares clave-valor en el Map.

    Iteración:

4. Puedes iterar sobre los elementos de un Map usando un bucle for...of, y acceder a las claves, valores o a ambos.

    Por ejemplo, con el método entries() se obtiene un iterador sobre los pares clave-valor.

#### Diferencias entre un Map y un objeto en JavaScript:
1. Claves: En un objeto, las claves son siempre cadenas o símbolos, pero en un Map, las claves pueden ser de cualquier tipo (incluso objetos).

2. Orden de inserción: Los objetos no garantizan el orden de las claves (aunque en la mayoría de los casos modernas parece estar ordenado), mientras que un Map preserva el orden exacto en que se insertan las claves.

3. Rendimiento: Los Map suelen ser más eficientes en operaciones de inserción, eliminación y búsqueda, especialmente cuando tienes muchas claves, ya que están optimizados para eso.

En resumen, el Map es una estructura de datos muy útil cuando necesitas asociar claves con valores y mantener un orden de inserción preciso o usar claves no tradicionales.

## Usando un array como llave

```javascript

// Crear un nuevo Map
let mapa = new Map();

// Crear un array como clave
let claveArray = [1, 2, 3];

// Establecer el valor para ese array como clave
mapa.set(claveArray, 'valor asociado');

// Obtener el valor usando el mismo array como clave
console.log(mapa.get(claveArray)); // 'valor asociado'

```
