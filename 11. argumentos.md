# Argumentos en las funciones

La palabra 'arguments' solo existe en las funciones regulares (Expression) y declaración (Declaration) de funciones

## Function Expression
Este tipo de funciones cuentan con su propia palabra 'arguments'

Y se pueden pasar varios argumentos al mismo tiempo
```javascript
const sumar = function (a, b){

    console.log(arguments) // Imprime un array de los argumentos al momento de llamar la función
    return a + b;
}
sumar (2, 2);
sumar(2, 2, 5, 5);

// Aquí se imprimen los argumentos en un array
```

## Arrow Functions

```javascript
var sumar2 = (a, b) =>{

    console.log(arguments) // Imprime un array de los argumentos al momento de llamar la función
    return a + b;
}
sumar2 (2, 2, 8, 8)
// Arroja un error de que la palabra 'arguments' no está definida

```
