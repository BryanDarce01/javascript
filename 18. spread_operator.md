## . . . ( 3 puntos )

# Se le llama Spread a los 3 puntos que están después del lado derecho del signo igual. 

# Se le llama REST a los 3 puntos que están del lado izquierdo antes del signo igual.

Lo que hace el operador Spread es sacar todos los valores del array que se está llamando y escribirlos de manera individual en el nuevo array.

### Regla 1. Este elemento se parece a la destructuración de objetos pero la diferencia radica en que este saca todos los elementos del array y no crea nuevas variables y lo usamos para escribir valores separados por coma.

### Regla 2.  Este elemento solo lo podemos usar cuando creamos nuevos arreglos o cuando pasamos valores a una función.

Porque aquí si se esperan varios valores separados por coma.

### Regla 3. Solo se puede usar con elementos que se demoniman iterables que son: Arreglos, Strings, Maps y Sets. No en objetos.

#### Actualización, a partir del 2018 se puede usar el operador spread en objetos.

```javascript

const array = [ 4, 5, 6 ];

const nuevoArray = [ 1, 2, 3 ...array];

console.log(nuevoArray);

//Resultado: [1, 2, 3, 4, 5, 6]

```

Sino le podemos los 3 puntos, lo que sucedería es que escribiría el el array completo sin sacar los elementos.

```javascript

const array = [ 4, 5, 6 ];

const nuevoArray = [ 1, 2, 3 array];

console.log(nuevoArray);

//Resultado: [1, 2, 3, Array(3)]

```

Para imprimir en la consola todos los elementos sin los [ ] (los corchetes), debemos usar el operador spread.

```javascript

const array = [ 4, 5, 6 ];

const nuevoArray = [ 1, 2, 3 array];

console.log(...nuevoArray);

//Resultado: 1, 2, 3, 4, 5, 6

```

Podemos crear copias de un array sin modificar el original.

```javascript

const restaurant = {
  name: 'Classico Italiano',
  location: 'Via Angelo Tavanti 23, Firenze, Italy',
  categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
  starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
  mainMenu: ['Pizza', 'Pasta', 'Risotto']

}

const copiaMenu = [...restaurant.mainMenu];

console.log(copiaMenu);
// Resultado: ["Pizza", "Pasta", "Risotto"]
```

Podemos unir 2 arrays.

```javascript

const restaurant = {
  name: 'Classico Italiano',
  location: 'Via Angelo Tavanti 23, Firenze, Italy',
  categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
  starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
  mainMenu: ['Pizza', 'Pasta', 'Risotto']

}

const copiaMenu = [...restaurant.mainMenu, ...restaurant.starterMenu];

console.log(copiaMenu);
// Resultado: ["Pizza", "Pasta", "Risotto", "Focaccia", "Bruschetta", "Garlic Bread", "Caprese Salad" ]

```

### Ejemplo de iterar en un string

```javascript

const nombre = 'Bryan';
const nuevo = [...nombre];
console.log(nuevo);

// Resultado: ["B", "r", "y", "a", "n"]

```

Si queremos sacar todos los elementos del array.

```javascript

const nombre = 'Bryan';
const nuevo = [...nombre];
console.log(...nuevo);

// Resultado: B r y a n

```

### No funciona en objetos

No funciona porque la parte de '${ }' no espera múltiple valores separados por coma.
```javascript

const nombre = 'Bryan';

console.log(`Mi nombre es ${...nombre}`);

// Resultado: error

```

### Usando los 3 puntos en funciones

```javascript

const restaurant = {
  name: 'Classico Italiano',
  location: 'Via Angelo Tavanti 23, Firenze, Italy',
  categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
  ordenarPasta: function(ingrediente1, ingrediente2){

    console.log(`Pasta con ${ingrediente1} y ${ingrediente2}`);
  }
}

const ingredientes = [
    promt("Ingrese el ingrediente 1"),
    promt("Ingrese el ingrediente 2"),
]

restaurant.ordenarPasta(...ingredientes);

// Resultado: Pasta con ingrediente1 y ingrediente2

// Con la palabra promt le sale una ventana flotante al usuario y ahí escribe sus ingredientes

```

### En objetos el spread operator

```javascript

const restaurant = {
  fundador: 'Bryan',
  name: 'Classico Italiano',
  location: 'Via Angelo Tavanti 23, Firenze, Italy',
  categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic']
}

const copiaRestaurante = [ ...restaurant];

// Cambio del nombre del fundador en la copia

copiaRestaurante.fundador = 'Pedro';

console.log(copiaRestaurante.fundador);
console.log(restaurant.fundador)

// Resultado:
// Pedro
// Bryan

```

# Spread Operator with Functions

## REST Parameters

```javascript

const sumarValores = function (...numeros){

  let sumar = 0;
  for (let i = 0; i < numeros.lenght; i++) sumar += numeros[i];

  console.log(sumar);

}


sumarValores(2, 3);

// Resultado: 5

sumarValores(5, 5, 5, 5);

// Resultado 20

```

También podemos usar el operador Spread para pasar una variable que sea un array y usando los 3 puntos, pasarlo como argumento en una función.

Aquí lo que hacemos es decirle a la función que debe separar todos los elementos del arreglo con una coma, y así ya pueden entrar a la función.

```javascript

const x = [ 1, 2, 3];

sumarValores(...x)

```

### Usando el operador REST podemos hacer que un parámetro sea el principal y el resto opcionales, y con los opcionales nos devuele un array vacío sino se declaran.
### Sino podemos trabajar con los elementos del array.

```javascript

const restaurant = {
  name: 'Classico Italiano',
  location: 'Via Angelo Tavanti 23, Firenze, Italy',
  categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
  ordenarPizza: function(ingredientePrincipal, ...otrosIngredientes) {

  console.log(ingredientePrincipal);
  console.log(otrosIngredientes);
  }
}

restauran.ordenarPizza('jamón', 'pepperoni', 'salami');
// Resultado: jamón, (3)["pepperoni", "salami"]

```

En este caso, si solo le pasamos 1 argumento

```javascript

restauran.ordenarPizza('jamón');
// Resultado: jamón, []

```
