# THIS Keyword
## The THIS kerword in a global scope a the window object

## Regular Functions
In Regular Functions de THIS keyword will be undefined because de strict mode

En este tipo de funciones se crea su propia palabra THIS y por eso al momento de querer acceder a esta sale como Undefined sino está creada

## Function Expression
Este tipo de funciones crean su propia palabra THIS y se puede acceder a esta

```javascript
cons jonas = {
    year: 1991,
    calcAge: function(){
        console.log(this);
        console.log(2024-this.year);
        }
};
jonas.calcAge();
// Aquí se imprime el resultado
```

## La palabra THIS apunta al método que la está llamano y por eso se pueden copiar los elementos y se le llama

En este ejemplo de códgio lo que hace es que matilda está copiando la función de calcAge del objeto jonas, y como cada objeto crea su propia palabra THIS al usar Functions Expressions, lo que hace es apuntar al objeto matilda y buscar la palabra year dentro del objeto que la llama.

No va a buscar la palabra THIS del objeto jonas

### Prestar
```javascript

const matilda = {
    year: 2017,
};

matilda.calcAge = jonas.calcAge; // En este pedazo solo se pone calcAge sin los paréntesis porque así es como se copian los métodos, si se le pone el paréntesis lo que se hace es que ahí se está llamando a la función pero no copiando.
matilda.calcAge();
```

Si se intenta copiar el método calcAge a una función, va a dar error porque una función regular o Expression si tienen su propia palabra THIS.

```javascript
const prueba = jonas.calcAge;

prueba();
// Aquí se intenta buscar a la palabra year que existe el emétodo jonas pero como la variable "prueba" no tiene creada la palabra "year", es por eso que da error

```

## Arrow Functions
The arrow functions do not get it´s own THIS keyword. It use the lexical THIS keyword (of the parent function or global scope)

# No usar VAR
Porque var crea sus propiedades en el Global Scope y puede generar bugs
# No Arrow Functions como métodos

```javascript
var nombre = 'Juan'

cons pedro = {
    nombre : 'Pedro'
    year: 1991,
    calcAge: function(){
        console.log(this);
        console.log(2024-this.year);
        },
    // Usando arrow function como método    
    saludo: () => {
        console.log(`Hola ${nombre}`)
    }
};
pedro.saludo();
// Hola Juan
```
En el ejemplo de arriba sale Juan porque 'pedro' es un objeto y no una función y por eso el toma la variable nombre de Global Scope que es 'Juan'
