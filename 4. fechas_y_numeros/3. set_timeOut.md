# setTimeout

Es una función que se usa para ejecutar un código o llamar a una función después de un período de tiempo especificado (en milisegundos).

## 🔧 Sintaxis

```javascript

setTimeout(función, tiempoEnMs, arg1, arg2, ...);

```

- `función` → la función que quieres ejecutar.
- t`iempoEnMs` → cuánto esperar antes de ejecutarla (en milisegundos).
- `arg1, arg2,` ... → argumentos opcionales que se pasan a la función.

## Ejemplos
1. Con función anónima

```javascript

setTimeout(() => {
  console.log("Hola después de 2 segundos");
}, 2000);

```

⏳ Espera 2 segundos y luego muestra el mensaje.

## 2. Llamando a una función existente

```javascript

function saludar(nombre) {
  console.log(`Hola, ${nombre}`);
}

setTimeout(saludar, 3000, "Bryan");
// Después de 3 segundos: "Hola, Bryan"

```

## 3. Guardando el ID para cancelarlo

```javascript

let timerId = setTimeout(() => {
  console.log("Esto nunca se ejecutará");
}, 5000);

clearTimeout(timerId); // Cancela la ejecución

```
