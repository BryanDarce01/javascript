# setInterval

`setInterval()` es una función que ejecuta otra función o código de forma repetida, cada cierto intervalo de tiempo (en milisegundos), hasta que lo detengas usando `clearInterval()`.

## 🔧 Sintaxis

```javascript

let id = setInterval(función, tiempo, arg1, arg2, ...);

```

- `función` → La función que quieres ejecutar.
- `tiempo` → Intervalo en milisegundos (1000 ms = 1 segundo).
- `arg1, arg2, ...` (opcional) → Argumentos que se le pasan a la función.

## Ejemplo básico

```javascript

function saludar() {
  console.log("Hola cada 2 segundos");
}

let intervalo = setInterval(saludar, 2000); // 2000 ms = 2 segundos

```

Esto imprimirá "Hola cada 2 segundos" indefinidamente.

## Detener un setInterval()

```javascript

clearInterval(intervalo);

```

### Ejemplo con parada después de 6 segundos:

```javascript

let contador = 0;

let intervalo = setInterval(() => {
  contador++;
  console.log(`Ejecución #${contador}`);
  
  if (contador === 3) {
    clearInterval(intervalo);
    console.log("Intervalo detenido");
  }
}, 2000);

```

### 📌 Esto imprime:

```javascript

Ejecución #1
Ejecución #2
Ejecución #3
Intervalo detenido

```

## Pasar parámetros

```javascript

function mostrarMensaje(nombre) {
  console.log(`Hola, ${nombre}`);
}

let intervalo = setInterval(mostrarMensaje, 1000, "Bryan");

```

## Diferencia con setTimeout()

- `setTimeout()` → Se ejecuta una vez, después del tiempo especificado.

- `setInterval()` → Se ejecuta repetidamente cada cierto tiempo, hasta que se detenga.

# IMPORTANTE

**Es mejor un intervalo que respete el tiempo de ejecución de la función (evitando superposición), es mejor usar `setTimeout()` recursivo en lugar de `setInterval()`.**
