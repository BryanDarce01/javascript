# Fechas

## 📅 Crear una fecha (new Date())
1. Fecha y hora actual

```javascript

const ahora = new Date();
console.log(ahora);

```

## 2. Desde una cadena

```javascript

const fecha = new Date("2025-08-06T10:30:00");

```

## 3. Desde valores (año, mes, día, etc.)

```javascript

const fecha = new Date(2025, 7, 6, 10, 30, 0); // Mes 7 = agosto (0-indexado)

```

## 4. Desde timestamp (milisegundos desde 1970-01-01)

```javascript

const fecha = new Date(0); // Epoch
const fecha2 = new Date(1628160000000); // milisegundos

```
## 🧾 Métodos útiles de lectura (get)

```javascript

const fecha = new Date();

fecha.getFullYear();     // 2025
fecha.getMonth();        // 7 (agosto, empieza desde 0)
fecha.getDate();         // 6
fecha.getDay();          // 3 (miércoles, 0 = domingo)
fecha.getHours();        // 10
fecha.getMinutes();      // 30
fecha.getSeconds();      // 0
fecha.getMilliseconds(); // 0
fecha.getTime();         // timestamp (ms desde 1970)

```

## ✍️ Métodos para cambiar fecha (set)

```javascript

fecha.setFullYear(2030);
fecha.setMonth(0);         // enero
fecha.setDate(1);          // día
fecha.setHours(8);
fecha.setMinutes(45);

```

## 🕗 Formatos comunes

```javascript

fecha.toString();          // "Wed Aug 06 2025 10:30:00 GMT..."
fecha.toDateString();      // "Wed Aug 06 2025"
fecha.toTimeString();      // "10:30:00 GMT..."
fecha.toISOString();       // "2025-08-06T10:30:00.000Z"
fecha.toLocaleDateString(); // según el idioma

```

## 📏 Comparar fechas

```javascript

const f1 = new Date("2025-08-06");
const f2 = new Date("2025-08-07");

console.log(f1 < f2);  // true
console.log(f1.getTime() === f2.getTime()); // false

```

## 🧮 Diferencia entre fechas

```javascript

const inicio = new Date("2025-08-01");
const fin = new Date("2025-08-06");

const msDiff = fin - inicio; // diferencia en milisegundos
const dias = msDiff / (1000 * 60 * 60 * 24); // convertir a días
console.log(dias); // 5

```

## 🔧 Otros métodos útiles
- `Date.now()` → timestamp actual (en ms):

```javascript

const ahora = Date.now();

```

`Date.parse("2025-08-06")` → convierte a timestamp:

```javascript

const t = Date.parse("2025-08-06");

```
