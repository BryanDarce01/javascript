# Números

## ✅ Math.ceil()

- Redondea hacia arriba al entero más cercano.

``` javascript

Math.ceil(4.2);   // 5
Math.ceil(-4.2);  // -4

```

## ✅ Math.floor()

- Redondea hacia abajo al entero más cercano.

``` javascript

Math.floor(4.8);   // 4
Math.floor(-4.8);  // -5

```

## ✅ Math.round()

- Redondea al entero más cercano (normal).

```javascript

Math.round(4.5);  // 5
Math.round(4.4);  // 4

```

## ✅ Math.trunc()

- Elimina los decimales sin redondear (trunca).

```javascript

Math.trunc(4.9);   // 4
Math.trunc(-4.9);  // -4

```

## ✅ Math.max()

- Devuelve el número más grande entre varios.

```javascript

Math.max(1, 5, 10, 2);  // 10

```
## ✅ Math.min()

- Devuelve el número más pequeño entre varios.

```javascript

Math.min(1, 5, 10, 2);  // 1

```

## ✅ toFixed()

- Devuelve un string con un número fijo de decimales.

```javascript

let num = 5.6789;
num.toFixed(2);  // "5.68" ← string

```

### Si necesitas que vuelva a número:

```javascript

parseFloat(num.toFixed(2)); // 5.68

```

## ✅ Number.isInteger()

- Verifica si un número es entero.

```javascript

Number.isInteger(5);    // true
Number.isInteger(5.1);  // false

```

## ✅ Number.parseInt() y Number.parseFloat()

- Convierte strings a números (enteros o flotantes):

```javascript

Number.parseInt("10.99");   // 10
Number.parseFloat("10.99"); // 10.99

```

## ✅ Math.pow(base, exponente)

- Potencia:

```javascript

Math.pow(2, 3); // 8
// o más moderno:
2 ** 3; // 8

```

## ✅ Math.sqrt()

- Raíz cuadrada:

```javascript

Math.sqrt(16); // 4

```

## ✅ Math.random()

- Devuelve un número aleatorio entre 0 y 1:

```javascript

Math.random(); // Ej: 0.62848

// Aleatorio entre 1 y 10:
Math.floor(Math.random() * 10) + 1;

```

## 🧠 ¿Cuándo usar cuál?

| Método         | Uso principal                          |
| -------------- | -------------------------------------- |
| `ceil()`       | Redondear hacia arriba                 |
| `floor()`      | Redondear hacia abajo                  |
| `round()`      | Redondear normalmente                  |
| `trunc()`      | Quitar los decimales                   |
| `toFixed(n)`   | Formatear a `n` decimales (como texto) |
| `max(), min()` | Obtener el mayor o menor valor         |
| `random()`     | Generar números aleatorios             |



# Remainder Operator

Se llama así porque devuelve el residuo de una división **(también se le conoce como operador módulo o modulus).**

## ✅ Sintaxis

```javascript

a % b

```

Devuelve el resto de dividir `a entre b.``

## 🔍 Ejemplos básicos

```javascript

10 % 3  // 1   → porque 3 * 3 = 9, y sobra 1
8 % 2   // 0   → porque 8 se divide exacto entre 2
7 % 4   // 3   → porque 4 * 1 = 4, y sobra 3

```

# 🧠 ¿Para qué sirve?
- Saber si un número es par o impar:

```javascript

let num = 6;
console.log(num % 2 === 0 ? "Par" : "Impar"); // "Par"

```
- Ejecutar algo cada n ciclos:

```javascript

for (let i = 1; i <= 10; i++) {
  if (i % 3 === 0) {
    console.log(i + " es múltiplo de 3");
  }
}

```

#### ⚠️ Nota sobre negativos

```javascript

-7 % 3  // -1
7 % -3  // 1

```

El signo del resultado sigue el primer número (`a % b` → el signo sigue a `a`).

## 🆚 No confundir con Math.floor()

`a % b` devuelve el residuo.

`Math.floor(a / b)` devuelve cuántas veces cabe `b` en `a`.

# BigInt

Es un tipo de dato en JavaScript que permite trabajar con números enteros muy grandes, más allá del límite que puede manejar el tipo `Number`.

## 🧠 ¿Por qué existe BigInt?
En JavaScript, el tipo Number puede representar enteros seguros solo hasta:

```javascript

2^53 - 1 = 9007199254740991 (Number.MAX_SAFE_INTEGER)

```

Más allá de eso, los cálculos pueden volverse **imprecisos**.

## ✅ ¿Qué es BigInt?

BigInt permite trabajar con enteros arbitrariamente grandes, sin perder precisión.

## ✨ Sintaxis

- Usando el sufijo n:

```javascript

const big = 123456789012345678901234567890n;

```

- O con el constructor:

```javascript

const big = BigInt("123456789012345678901234567890");

```

## 📌 Operaciones con BigInt

```javascript

const a = 12345678901234567890n;
const b = 9876543210n;

console.log(a + b);  // suma
console.log(a - b);  // resta
console.log(a * b);  // multiplicación
console.log(a / b);  // división (¡sin decimales!)
console.log(a % b);  // módulo

```

## ⚠️ Restricciones

## ❌ No se puede mezclar BigInt con Number directamente:

```javascript

const a = 10n;
const b = 5;
console.log(a + b); // ❌ TypeError

```

Debes convertir uno de los dos:

```javascript

console.log(a + BigInt(b)); // ✅

```


## ❌ No funciona con métodos de Math:

```javascript

Math.sqrt(16n); // ❌ TypeError

```

## 📤 Conversión

```javascript

const a = BigInt(100);
const b = Number(100n);

```

## 🧪 Comparación

```javascript

10n === 10      // false (distinto tipo)
10n == 10       // true  (comparación suelta)

```
