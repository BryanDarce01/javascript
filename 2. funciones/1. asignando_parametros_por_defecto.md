# Parámetros por defecto

Aquí estamos asignando valores por defecto, sino se pasan los argumentos.

```javascript

function vuelos(numPasajeros = 0, precio = 0, destino = 'ninguno') {
    const precioTotal = numPasajeros * precio;
    const mensaje = `El precio total de ${numPasajeros} pasajeros a ${destino} es de $${precioTotal}`;
    // return mensaje;
    console.log(mensaje);

}

vuelos(5, 100, 'Lima'); // 'El precio total de 5 pasajeros a Lima es de $500'

vuelos(3, 200, 'Madrid'); // 'El precio total de 3 pasajeros a Madrid es de $600'

vuelos(); // 'El precio total de 0 pasajeros a ninguno es de $0'

```
