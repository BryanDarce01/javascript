# IIFE (Immediately Invoked Function Expression)

## Forma 1

``` javascript

(function() {
    console.log('Solo una vez se puede ejecutar este script')
}
)()// Con estos últimos paréntesis se ejecuta la función inmediatamente
// Esto es un IIFE (Immediately Invoked Function Expression)

```

## Forma 2 con funciones de flecha

``` javascript

(() => {
    console.log('También se puede hacer con una función flecha')
})() // También un IIFE con función flecha

```
