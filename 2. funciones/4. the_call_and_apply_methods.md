# Manipular de manera manual la palabra THIS

### Creamos el objeto **avianca** y usamos la palabra clave **this** dentro del método para referirnos al propio objeto.

#### Explicación:
**this** dentro del método agregarReserva hace referencia al objeto avianca, ya que el método se está invocando directamente sobre él (avianca.agregarReserva(...)).

Esto permite acceder a las propiedades del objeto como this.nombre y this.id.

No estamos manipulando this de forma "manual" como con .call(), .apply() o .bind(), sino que se está utilizando de manera implícita y correcta dentro del contexto del objeto.

``` javascript
const avianca = {
    nombre: "Avianca",
    id: "AV",
    reservas: [],
    agregarReserva(numReserva, nombrePasajero) {
        console.log(`${nombrePasajero} ha reservado un vuelo con ${this.nombre} con el número de reserva ${this.id}${numReserva}.`);
        this.reservas.push({ vuelo: `${this.id}${numReserva}`, nombrePasajero });
    }
};

avianca.agregarReserva(123, "Juan Pérez");

// 'Juan Pérez ha reservado un vuelo con Avianca con el número de reserva AV123.'

avianca.agregarReserva(456, "María López");

//'María López ha reservado un vuelo con Avianca con el número de reserva AV456.'

console.log(avianca.reservas);

//[ { vuelo: 'AV123', nombrePasajero: 'Juan Pérez' }, { vuelo: 'AV456', nombrePasajero: 'María López' } ]

```

## CALL()

``` javascript

const aeroMexico = {
    nombre: "AeroMéxico",
    id: "AM",
    reservas: []
};

const agregarReservaAM  = avianca.agregarReserva

agregarReservaAM.call(aeroMexico, 789, "Carlos Sánchez");

// 'Carlos Sánchez ha reservado un vuelo con AeroMéxico con el número de reserva AM789.'

agregarReservaAM.call(avianca, 101, "Ana García");

// 'Ana García ha reservado un vuelo con Avianca con el número de reserva AV101.'

```
🧠
### Explicación:
**agregarReservaAM** es una referencia a la función **agregarReserva** del objeto avianca. Esta función usa **this** internamente.

Al usar **.call(aeroMexico, ...)**, estás cambiando manualmente el contexto de **this** dentro de la función para que apunte al objeto **aeroMexico**.

Esto significa que **this.nombre, this.id, etc.**, dentro de **agregarReserva**, harán referencia a las propiedades del objeto aeroMexico.

👓 Para que funcione correctamente, debes asegurarte de que aeroMexico esté definido con propiedades similares.

## APPLY

.apply() también ejecuta la función inmediatamente.

**Los argumentos se pasan como un array.**

``` javascript

const vivaAerobus = {
    nombre: "Viva Aerobus",
    id: "VB",
    reservas: []
};

const reservaViva = [555, 'Luis Martínez'];
agregarReserva.apply(vivaAerobus, reservaViva);

// 'Luis Martínez ha reservado un vuelo con Viva Aerobus con el número de reserva VB555.'

agregarReserva.apply(vivaAerobus, [456, "Ana Torres"]);

// 'Ana Torres ha reservado un vuelo con Viva Aerobus con el número de reserva VB456.'

```

## IMPORTANTE

### También podemos usar el operador spread con el método .call()
### En JavaScript moderno, es común preferir .call() sobre .apply(), ya que con el operador spread (...) podemos pasar los argumentos fácilmente como si fueran individuales.

``` javascript

agregarReserva.call(vivaAerobus, ...reservaViva);

// 'Luis Martínez ha reservado un vuelo con Viva Aerobus con el número de reserva VB555.'

```

🧠 
### Explicación:
**.apply()** solía ser útil cuando tenías un array de argumentos.

Pero ahora, con el spread operator (...), puedes usar .call() con arrays también, haciendo a .apply() menos necesario.
