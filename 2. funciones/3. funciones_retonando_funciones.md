# Funciones que retornan otras funciones

Primero se declara la función **toma_de_valores** con sus parámetros.

Después se crea la función **resta**, que también tiene sus propios parámetros.

La ejecución se realiza en el mismo orden.

En la variable **resultado** se guarda el retorno de la función **toma_de_valores**, que a su vez es una función almacenada en la variable **valores**.

Una vez que se toman los argumentos de la primera función (almacenados en la variable **resultado**), se ejecuta la segunda función, que es **resta**.

```javascript

const valores = function toma_de_valores(valo_a, valor_b) {
  return function resta(a, b) {
    console.log(`Valores de entrada: ${valo_a}, ${valor_b}`);
    console.log(`Resultado de la resta: ${a - b}`);
  }
}

const resultado = valores(2, 10);
resultado(5, 1);
// Valores de entrada: 2, 10
// Resultado de la resta: 4

```


# Función flecha que retorna otra función

En este ejemplo, `sumarArrow` es una función flecha que retorna otra función.

Cada función toma un argumento. La primera recibe `valor_1`, y la segunda recibe `valor_2`. Al final, se imprime la suma de ambos valores.

```javascript

const sumarArrow = valor_1 => 
  valor_2 => 
    console.log(`Primer valor: ${valor_1}, Segundo valor: ${valor_2} y el resultado es: ${valor_1 + valor_2}`);

sumarArrow(5)(10);s
// Primer valor: 5, Segundo valor: 10 y el resultado es: 15

```
