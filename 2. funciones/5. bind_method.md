# MÉTODO BIND

Al igual que el método **.call()**, el método **.bind()** nos permite manipular la palabra **this**.

La diferencia es que **.bind()** no ejecuta la función inmediatamente, sino que devuelve una nueva función en la que **this** queda permanentemente vinculado al objeto que especifiquemos.

En este ejemplo usamos la función **agregarReserva** con el método **bind()** para que la palabra **this** apunte al objeto  **avianca**.

``` javascript

const avianca = {
    nombre: "Avianca",
    id: "AV",
    reservas: [],
    agregarReserva(numReserva, nombrePasajero) {
        console.log(`${nombrePasajero} ha reservado un vuelo con ${this.nombre} con el número de reserva ${this.id}${numReserva}.`);
        this.reservas.push({ vuelo: `${this.id}${numReserva}`, nombrePasajero });
    }
};


const aviancaBIND = avianca.agregarReserva.bind(avianca);

aviancaBIND(456, "María López");

// 'María López ha reservado un vuelo con Avianca con el número de reserva AV456.'

```

## 1. Aplicación Parcial
### Partial Application
### Ejemplo1
Chat GPT:

La aplicación parcial es una técnica de programación funcional en la que una función que espera varios argumentos puede ser preconfigurada con algunos de ellos, devolviendo una nueva función que espera el resto.

 Es como rellenar parcialmente un formulario y guardarlo para completarlo más tarde.

Lo que hacemos en este ejemplo con el objeto **avianca** es usar el método **.bind()** para predefinir manualmente el número de reserva **(numReserva)** como **"555"**.
De esta manera, la nueva función **reservarConAvianca** solo necesita recibir el nombre del pasajero.

 ``` javascript

const avianca = {
    nombre: "Avianca",
    id: "AV",
    reservas: [],
    agregarReserva(numReserva, nombrePasajero) {
        console.log(`${nombrePasajero} ha reservado un vuelo con ${this.nombre} con el número de reserva ${this.id}${numReserva}.`);
        this.reservas.push({ vuelo: `${this.id}${numReserva}`, nombrePasajero });
    }
};

const reservarConAvianca = avianca.agregarReserva.bind(avianca, "555");

reservarConAvianca("Carlos Pérez");

// 'Carlos Pérez ha reservado un vuelo con Avianca con el número de reserva AV555.'

```

✅ 

### ¿Qué logramos?

1. Usamos aplicación parcial para fijar el valor de numReserva a "555".

2. La función resultante solo requiere un argumento: **nombrePasajero**.

3. **this** sigue apuntando correctamente al objeto avianca gracias al primer argumento de .bind().


### Ejemplo2

Aquí estamos predefiniendo el primer argumento (impuesto = 0.20) usando **.bind()**.

Aunque impuestoGeneral es una función flecha (sin this), **.bind()** aún puede usarse para hacer aplicación parcial si solo te interesa fijar uno o más argumentos.

🔧 null se pasa como primer parámetro porque **bind** normalmente fija el valor de this, pero las arrow functions no usan su propio this, así que puedes poner null o cualquier cosa ahí, y no afectará.

En resumen, pasamos la palabra **null** porque en este caso no se necesita manipular la palabra **this**.
 ``` javascript

const impuestoGeneral =(impuesto, precioProducto)  => precioProducto + precioProducto * impuesto;
console.log(impuestoGeneral(0.10, 100));

const impuesto20Porciento = impuestoGeneral.bind(null, 0.20);
console.log(impuesto20Porciento(100)); // 120
console.log(impuesto20Porciento(200)); // 240

  ```


## 2. Con Event Listener

Lo que hacemos aquí es que al momento de dar 'click' al botón de 'comparAvion', este incremente su valor.

 ``` javascript

aeroMexico.aviones = 100;
aeroMexico.comprarAvion = function() {
    console.log(this),
    this.aviones++;
    console.log(`AeroMéxico ahora tiene ${this.aviones} aviones.`); 
};

document
    .querySelector('.buy')
    .addEventListener('click', aeroMexico.comprarAvion.bind(aeroMexico)
    );

// {nombre: "AeroMexico", id: "AM", reservaas: Array(), aviones: 100, 101}

 ```
