# Logical Assigment Operators

### 1. Usando Short Circuiting
#### Evaluación de Cortocircuito


### Vamos a crear 2 objetos que no tengan un método y con la evaluación de cortocircuito, devuelve el primer valor no Nulo

```javascript

const restaurant = {
    name: 'Classico Italiano',
    numGuest: '20'
  
  }


const restaurant1 = {
    name: 'Classico Italiano',
    fundador: 'Pedro'

  }

```

#### Lo que hacemos aquí es evaluar si existe el método 'numGuest' en ambos objetos, y sino existen, que les asigne el valor 10

ESte operador lo que hace es evaluar si la primera condición es verdadera, y la devuelve, sino devuelve la segunda.

```javascript

restaurant.numGuest = restaurant.numGuest || 10;

restaurant1.numGuest = restaurant1.numGuest || 10;


console.log(restaurant); // Resultado:  {name:  'Classico Italiano', numGuest: '20'}
console.log(restaurant1); // Resultado: {name: 'Classico Italiano', fundador: 'Pedro', numGuest: '10'}
```

### Usando Logical Assigment

El resultado es el mismo que el código de arriba.

```javascript

restaurant.numGuest ||= 10;
restaurant1.numGuest ||= 10;

console.log(restaurant);
console.log(restaurant1);

// Resultado:  {name:  'Classico Italiano', numGuest: '20'}
// Resultado: {name: 'Classico Italiano', fundador: 'Pedro', numGuest: '10'}

```

### Si asignamos el valor 0 al método numGuest, obtenemos la asignación del valor 10 al método numGuest si usamos el operador OR

Ya que el valor 0 lo reconoce como Falsy así que asigna el valor 10 por defecto.

```javascript

const restaurant = {
    name: 'Classico Italiano',
    numGuest: '0'
  
  }

restaurant.numGuest ||= 10;

console.log(restaurant);

// Resultado:  {name:  'Classico Italiano', numGuest: '10'}

```

### Para este ejempo debemos usar Nullish assigment

```javascript

restaurant.numGuest ??= 10;

console.log(restaurant);
// Resultado:  {name:  'Classico Italiano', numGuest: '0'}

```

### Operador AND

El operador AND en JavaScript evalúa los operandos de izquierda a derecha. Devuelve el primer valor falsy que encuentra (como false, 0, null, undefined, NaN o una cadena vacía). Si todos los valores son truthy, devuelve el último valor evaluado.

En resumen:

Si encuentra un valor falsy, lo devuelve.

Si no encuentra ninguno, devuelve el último valor truthy.

En el primer ejemplo, no asigna nada porque el método no existe.

En el segundo, como la expresión es verdadera, asigna la palabra ANONIMO.

```javascript

const restaurant = {
    name: 'Classico Italiano',
    numGuest: '20'
  
  }


const restaurant1 = {
    name: 'Classico Italiano',
    fundador: 'Pedro'

  }

restaurant.fundador = restaurant.fundador &&= '<ANONIMO>';

// Resultado: {name: 'Classico Italiano', numGuest: '20'}

restaurant1.fundador = restaurant1.fundador &&= '<ANONIMO>';
// Resultado: {name: 'Classico Italiano', numGuest: '20', fundador: '<ANONIMO>'}

```
