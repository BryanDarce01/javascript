# Manipulación de Objetos

Este es el objeto producto
```javascript

const producto = {
    nombre: 'Tablet',
    precio: 50,
    disponible: false
    
}
```


Reescribir un valor

```javascript

producto.disponible = true

```

Agregar un método

```javascript

producto.imagen = 'imagen.jpg'

```


Eliminar un método

```javascript

 delete producto.imagen

```


Sirve para que no se pueda agregar, quitar o modificar los métodos de un objeto

```javascript

object.freeze(producto) 

```

Para sellar un objeto y que no se pueden agregar o eliminar propiedades pero si modificar las que ya tiene

```javascript

object.seal(producto) 

```

## Destructuring de 2 0 más objetos

Tenemos 2 objetos con el mismo nombre de una propiedad que en este caso es "nombre"
```javascript

const producto = {
    nombre: 'Tablet',
    precio: 50,
    disponible: false
    
}

const cliente = {
    nombre: 'Juan',
    premium: true,
    direccion: {
        calle: 'Calle Antigua'
    }

}
```

Al querer llamar esas variables, va a arrojar un error
```javascript
const { nombre} = producto
const { nombre} = cliente
console.log(nombre)
```

Aquí renombramos la propiedad "nombre" del objeto "cliente"
Para acceder a "calle" del array "dirección" usamos los ":" más las "{ }"
```javascript
const { nombre} = producto
const { nombre: nombreCliente, direccion: { calle }} = cliente
console.log(nombre);
console.log(nombreCliente)
```

# Unión de 2 o más objetos
Usamos " ... " (tres puntos) para pasar las propiedades de "producto" al objeto "cliente"
Se le llama Spread Operator
```javascript

const producto = {
    nombre: 'Tablet',
    precio: 50,
    disponible: false
    
}

const cliente = {
    nombre: 'Juan',
    premium: true,
    ...producto

}

console.log(cliente.precio);
```

### En este codigo que no tiene los " ... " (tres puntos) lo que hace es crear una copia
Lo que hace es que crea 2 propiedades una "producto" y "cliente"
```javascript
const nuevoObbjeto = {
    producto,
    cliente
    
}
console.log(nuevoObjeto)
```

## Para crear un nuevo objeto a partir de otro objeto
En este nuevo objeto, como los objetos anteriores tienen la propiedad nombre, lo que hace es que se reasigna la propiedad "nombre" y se queda con la última que sería la del "cliente.nombre"
```javascript
const nuevoObbjeto = {
    ...producto,
    ...cliente
    
}
console.log(nuevoObjeto)
```

### Para crear un nuevo objeto con menos código

```javascript
const nuevoObjeto = Object.assign(producto, cliente);
console.log(nuevoObjeto);
```
