# Desestructuración de objetos
## Objet destructuring


### Creamos el objeto restaurante
```javascript

'use strict';

// Data needed for first part of the section
const restaurant = {
  name: 'Classico Italiano',
  location: 'Via Angelo Tavanti 23, Firenze, Italy',
  categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
  starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
  mainMenu: ['Pizza', 'Pasta', 'Risotto'],

  openingHours: {
    thu: {
      open: 12,
      close: 22,
    },
    fri: {
      open: 11,
      close: 23,
    },
    sat: {
      open: 0, // Open 24 hours
      close: 23,
    },
  },
};

```

### Creamos 3 variables con los corchetes { }

```javascript

const {name, openingHours, categories} = restaurant;
console.log(name, openingHours, categories);

//Resultado: Classico Italiano
// {thu: {...}, fri: {...}, sat: {}}
// (4) ['Italian', 'Pizzeria', 'Vegetarian', 'Organic']

```

### Asignando nuevos nombres a los métodos dentro de los objetos

Si queremos que 'name' ahora sea 'restName'. 
Usamos 'name: restName'

```javascript

const { name: restName, openingHours: hours, categories: tags } = restaurant;
console.log(restName, hours, tags);

//Resultado: Classico Italiano
// {thu: {...}, fri: {...}, sat: {}}
// (4) ["Italian", "Pizzeria", "Vegetarian", "Organic"]

```

### Accediendo a métodos que no existen

```javascript

console.log(restaurant.menu);
// Resultado: undefined

```

### Podemos asignarle un valor por defecto
En este caso, que sea un array sin datos y lo podemos combinar con métodos que si existan en el objeto usando 'categories'

```javascript

const { menu = [], categories: tags = [] } = restaurant;

console.log(menu, tags);

// Resultado: 
// []
//(4) ["Italian", "Pizzeria", "Vegetarian", "Organic"]
```

### Mutando variables

Tenemos las variables 'a' y 'b' y queremos que esas variables hagan referencia a los métodos que están en el objeto.

Para esto debemos usar ( ) paréntesis

```javascript

let a = 1;
let b = 2;

const objetoNuevo = { a: 10, b: 20};

// Aquí usamos los paréntesis

({ a, b }) = objetoNuevo;

console.log( a, b);

//Resultado: 10, 20

```

## Objetos anidados

Acceder a las horas en que abre y cierra un restaurante.

En este caso, el resultado es un objeto con las variables 'open' y 'close'.
```javascript

const {sat} = restaurant.openingHours;

console.log(sat);

// Resultado {open: 0, close: 23 }

```

### Usamos la destructuración

```javascript

const {sat: { open, close }} = restaurant.openingHours;

console.log(open, close);

// Resultado: 0 23

```

También se le pueden asignar nombres a las palabras 'open' y 'close' 

```javascript

const {sat: { open: abierto, close: cerrado }} = restaurant.openingHours;

console.log(abierto, cerrado);

// Resultado: 0 23

```

## Destructurando argumentos que se pasan en una función.

En este ejemplo se van a pasar argumentos pero no van a ser los típicos parámetros, sino que se va a pasar solo un objeto usando '{ }' y vamos a aplicar la destucturación a este objeto.

```javascript

const restaurant2 = {
    categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
    ordenParaLlevar: function({startCategories, hora}) {

        console.log(`Orden de: ${this.startCategories[categories]} se entrega a las ${horas}`)
    }
}


restaurant2.ordenParaLlevar({startCategories: 2, horas:'3pm'});

// Resultado: Orden de Vegetarian se entrega a las 3pm

```

### También podemos poner por defectos los parámetros


```javascript

const restaurant2 = {
    categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],

    ordenParaLlevar: function({
        startCategories = 1, 
        horas = '00:00'}) {

        console.log(`Orden de: ${this.startCategories[categories]} se entrega a las ${horas}`)
    }
}


restaurant2.ordenParaLlevar({startCategories: 2, horas:'3pm'});

// Resultado: Orden de Vegetarian se entrega a las 3pm
