# Métodos y propiedades de `Response`

## Mio

Cuando se usa `fetch`, la función devuelve un objeto de tipo `Response`. 

Este objeto tiene propiedades y métodos que se puede usar para trabajar con 
la respuesta `HTTP`.

## Propiedades más usadas del `Response`:

- ``status`` → El código de estado HTTP (ej. ``200``, ``404``, ``500``).

- ``statusText`` → El texto asociado al estado (ej. ``OK``, ``Not Found``).

- ``ok`` → Booleano, true si el estado está en el rango **200–299**.

- ``url`` → La URL de la respuesta final (después de redirecciones).

- ``type`` → Tipo de la respuesta (``basic``, ``cors``, ``opaque``, ``default``).

- ``redirected`` → **Booleano**, indica si la respuesta vino de una redirección.

- ``headers`` → Un objeto `Headers` con los encabezados de la respuesta.

- ``body`` → Un `ReadableStream` con el cuerpo bruto de la respuesta.

- ``bodyUsed`` → **Booleano**, indica si ya se leyó el `body`.
