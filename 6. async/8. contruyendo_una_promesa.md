# ¿Qué es una Promesa?

Una **Promesa** es un objeto que representa la terminación (o el fracaso) eventual de una operación asíncrona.
Se crea con el constructor:

``` javascript

const promesa = new Promise((resolve, reject) => {
  // código que se ejecuta de inmediato
});

```

- Ese `(resolve, reject) => { ... }` es una **función ejecutora**.

# La función ejecutora

Cuando se crea una **promesa**, se le pasa una función ejecutora que recibe dos argumentos:

1. `resolve(valor)` → Se llama cuando la operación fue exitosa.

    - Pasa el valor de la promesa al siguiente `.then()`.

2. `reject(error)` → Se llama cuando la operación falló.

    - Pasa el error al `.catch()`.

``` javascript

const miPromesa = new Promise((resolve, reject) => {
  const exito = true;

  if (exito) {
    resolve("✅ Operación exitosa");
  } else {
    reject("❌ Ocurrió un error");
  }
});

// Usar la promesa
miPromesa
  .then(resultado => console.log(resultado)) // "✅ Operación exitosa"
  .catch(error => console.error(error))     // si hubiera fallado
  .finally(() => console.log("🔚 Promesa terminada"));

```
## Ejemplo con asincronía

``` javascript

const esperar = new Promise((resolve, reject) => {
  setTimeout(() => {
    resolve("⏰ Pasaron 2 segundos");
  }, 2000);
});

esperar.then(msg => console.log(msg));

```

- ⏳ Después de 2 segundos → `"⏰ Pasaron 2 segundos"`.

# Métodos Importantes

## Creación

``` javascript

const promesa = new Promise((resolve, reject) => {
  // Ejecuta algo async
  setTimeout(() => resolve("✅ Listo"), 1000);
});

```

## Métodos principales
1. **.then(onFulfilled, onRejected)**

    - Se ejecuta si la promesa fue resuelta `(resolve)`.

``` javascript

promesa.then(valor => console.log("Éxito:", valor));

```

2. **.catch(onRejected)**

    - Se ejecuta si la promesa fue rechazada `(reject)`.

3. **.finally(onFinally)**

Se ejecuta siempre, sin importar éxito o error.

``` javascript

promesa.finally(() => console.log("Promesa terminada"));

```

## Métodos estáticos

1. **Promise.all([...])**

     - Ejecuta varias promesas en paralelo → espera a que todas terminen.

        Si alguna falla → rechaza todo.

``` javascript

Promise.all([p1, p2, p3])
  .then(valores => console.log(valores)) // [resultado1, resultado2, resultado3]
  .catch(err => console.error("Falló:", err));

```

2. **Promise.allSettled([...])**

    - Devuelve el estado de **todas**, aunque alguna falle.

``` javascript

Promise.allSettled([p1, p2, p3])
  .then(resultados => console.log(resultados));

```

3. **Promise.race([...])**

    - Devuelve el resultado de la primera promesa que termine (**éxito o error**).

``` javascript

Promise.race([p1, p2])
  .then(r => console.log("Ganó:", r))
  .catch(e => console.error("Error rápido:", e));

```

4. **Promise.any([...])**

    - Devuelve la primera promesa que se cumpla con éxito.

        Si todas fallan → error `AggregateError`.

## Estado de una Promesa

Una promesa puede estar en 3 estados:

1. **pending** (pendiente, no ha terminado).

2. **fulfilled** (resuelta con éxito, se llama `resolve`).

3. **rejected** (rechazada con error, se llama `reject`).
