# ¿Qué es throw?

- La palabra clave `throw` se usa para lanzar un error manualmente.

- Cuando se lanza un error, el flujo normal del programa se interrumpe y salta al bloque `catch` más cercano (si existe).

```javascript

function dividir(a, b) {
  try {
    if (b === 0) {
    throw new Error("No se puede dividir entre cero ❌");
    return a / b;
    }
}
 catch (error) {
  console.error("Error capturado:", error.message);
    }
}

dividir(4, 0); // Error capturado: No se puede dividir entre cero ❌

```

# ¿Qué es `new Error`?

- Es un **objeto de error** nativo en JavaScript.

- Cuando se hace `new Error("mensaje")`, se crea un objeto que contiene:

    - `message`: el texto del error.

    - `name`: el tipo de error (por defecto `"Error"`).

    - `stack`: la traza del error (para depuración).

## ✅ Valores

```javascript

const err = new Error("Algo salió mal");
console.log(err.name);     // "Error"
console.log(err.message);  // "Algo salió mal"
console.log(err.stack);    // Traza de dónde ocurrió el error

```

## ✅ Ejemplo real

En el caso de `fetch`, como vimos antes:

`fetch` no lanza `error` en un `404`, pero podemos forzar uno:

```javascript

fetch("https://restcountries.com/v3.1/name/jajaja")
  .then(response => {
    if (!response.ok) {
      throw new Error(`HTTP Error! status: ${response.status}`);
    }
    return response.json();
  })
  .then(data => console.log(data))
  .catch(error => console.error("Capturado:", error.message));

```

## ❌ Salida

```javascript

'Capturado: HTTP Error! status: 404'

```

## Diferencia clave

- `throw` lanza cualquier cosa (string, número, objeto, error).

- `throw new Error("mensaje")` es la forma recomendada, porque muestra un objeto de error con más información `(message, name, stack)`.

## Aquí se mete el `throw new Error` en una función para solo mandarla a llamar donde se necesite, se ponen 2 parámetros que es la URL para la petición y el mensaje de error a mostrar

```javascript

const msgError = function(url, message) {
    return fetch(url)
      .then(response => {
        if (!response.ok) {
          throw new Error(`message: ${message}, status: ${response.status}`);
        }
        return response.json();
      }
    )
}
msgError(
    "https://restcountries.com/v3.1/name/jajaja", "No se encontró el país"
  )
  .catch(error => console.error("Capturado:", error.message));

  
```
