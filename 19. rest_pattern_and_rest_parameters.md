# The REST Pattern

### Destructuración

Este elemento sirve para compactar varios elementos en una sola variable.

Debemos poner el operador REST ( ... ) antes del sigo =

Lo que hacemos aaquí es declarar las variables 'a' y 'b' para los primeros dos numeros del arreglo y el resto de elementos, guardarlos en la variable 'numeros'.

```javascript

const [ a, b, ...numeros] = [ 1, 2, 3, 4, 5 ];
console.log(a, b, numeros);

// Resultado: 1, 2 (3) [3, 4, 5]

```

### Usando REST Pattern para acceder a los valores de un objeto

```javascript

const restaurant = {
  name: 'Classico Italiano',
  location: 'Via Angelo Tavanti 23, Firenze, Italy',
  categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
  starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
  mainMenu: ['Pizza', 'Pasta', 'Risotto']

}

const [pizza, , risotto, ...otros] = [...restaurant.mainMenu, ...restaurant.starterMenu];

console.log(pizza,risotto, ...otros);

// Resultado: Pizza, Risotto ["Focaccia", "Bruschetta", "Garlic Bread", "Caprese Salad"]

```
Si solo queremos imprimir la variable 'otros'.

```javascript

console.log(...otros);

```

### Destructuración en objetos

Creamos el objeto ¿restaurant' con el método 'openingHours' que es un objeto que tiene los horarios de apertura.

```javascript

const restaurant = {
  name: 'Classico Italiano',
  location: 'Via Angelo Tavanti 23, Firenze, Italy',
  categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
  starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
  mainMenu: ['Pizza', 'Pasta', 'Risotto'],

  openingHours: {
    thu: {
      open: 12,
      close: 22,
    },
    fri: {
      open: 11,
      close: 23,
    },
    sat: {
      open: 0, // Open 24 hours
      close: 23,
    },
  },
};

```

Guardamos la variable 'sat' que es la única que no es día entre semana y los demás días los agrupamos en la variable 'weekDays'

```javascript

const {sat, ...weekDays} = restaurant.openingHours;
console.log(weekDays);

// Resultado: { thu: { open: 12, close: 22 }, fri: { open: 11, close: 23 } }

```

Tenemos la función "SUMAR" y queremos pasarle múltiples argumentos y no solo a + b

```javascript

  function sumar(...numeros){
    let suma = 0;
    for(let i = 0; i < numeros.length; i++) 
        {suma += numeros[i]
        }
    console.log(suma)

  }

  sumar(2,2); // 4
  sumar(2,2,2,2); // 8

```
Podemos para usar array y con el operador SPREAD, lo pasamos en la función sumar de arriba.

```javascript

  const numeros = [5,5,5];

  sumar(...numeros) // 15

```
