# Cuando usar Arrays

```javascript

const codigo1 = ['Tarea1', 'Tarea2', 'Tarea3'];

```

- 1. Cuando se necesita una lista de valores ordenada y puede contener duplicados
- 2. Cuando se necesita manipular la data


# Cuando usar Sets

```javascript

const codigo1 = new Set(['Tarea1', 'Tarea2', 'Tarea3']);

```

- 1. Para trabajar con valores únicos
- 2. Cuando el rendimiento es importante
- 3. Para remover valores duplicados de los arrays

# Cuando usar Objects

- 1. Cuando se necesitar usar funciones como llaves y se llaman métodos

- 2. Cuando trabajamos con JSON

```javascript

const codigo1 = {
    tarea1: 'Ver codigo',
    tarea2: 'ver base de datos'
}

```

- 1. Cuando se necesita describir los valores usando llaves
- 2. Es fácil escribir y acceder a sus valores usando el punto `( . )` y corchetes `[ ]`

# Cuando usar Maps

## Cuando se necesite asignar una llave a un valor y esa llave no es un string

```javascript

const codigo1 = new Map([

    [tarea: 'Hacer el login',]
    [tarea2: 'ver la base de datos']
]);

```
- 1. Mejor rendimiento
- 2. Las llaves pueden tener como llave cualquier valor
- 3. Fácil de iterar
- 4. Tamaño fácil de calcular
