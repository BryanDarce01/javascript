# Desestructuración de arrays
## array destructuring

Es una característica introducida en ES6 que permite extraer valores de un array y asignarlos a variables de una manera más fácil y legible.

En lugar de acceder a los elementos del array utilizando índices, puedes "desestructurar" el array directamente en la declaración de las variables.

### Antes del array destructuring

```javascript

const arr = [2,3,4];
const a = arr[0];
const b = arr[1];
const c = arr[2];

```

### Con array destructuring
Estamos desempaquetango el array
1. Se puede declarar n variables al mismo tiempo

```javascript
const arr = [2,3,4];
const [x,y,z] = arr;
console.log(x,y,z);
```

### Accediendo a los valores dentro de un objeto

```javascript
'use strict';

// Data needed for first part of the section
const restaurant = {
  name: 'Classico Italiano',
  location: 'Via Angelo Tavanti 23, Firenze, Italy',
  categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
  starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
  mainMenu: ['Pizza', 'Pasta', 'Risotto'],
}

// Obtenemos el primer y el segundo valor de las categorias
const [first, second] = restaurant.categories;
console.log(first, second);
// Resultado: Italian Pizzeria
```

### Aquí queremos solo el primer y el tercer valor

```javascript

const [first, ,third] = restaurant.categories;
console.log(first, third)
// Resultado: Italian Vegetarian
```

### Si queremos los valores en orden diferente

```javascript
const [primero, segundo] = [second, first];
console.log(primero, segundo)
// Resultado: Pizzeria Italian

```

### Agregamos una función 'order' al objeto

```javascript
const restaurant = {
  name: 'Classico Italiano',
  location: 'Via Angelo Tavanti 23, Firenze, Italy',
  categories: ['Italian', 'Pizzeria', 'Vegetarian', 'Organic'],
  starterMenu: ['Focaccia', 'Bruschetta', 'Garlic Bread', 'Caprese Salad'],
  mainMenu: ['Pizza', 'Pasta', 'Risotto'],
  
  order : function (starterIndex, mainIndex) {

    return [this.starterMenu[starterIndex], this.mainMenu[mainIndex]];
  },
}

console.log(restaurant.order(2, 0));
// Resultado: ["Garlic Bread", "Pizza]

```

### Empezamos con la desestructturación
Recibimos 2 valores retornados de una función dentro de un objeto

```javascript
cons [opcion1, opcion2] = restaurant.order(2, 0);
console.log(opcion1, opcion2);

```

### Datos anidados

```javascript

const nested = [2, 4, [5, 6]];
const [a, , b] = nested;
console.log(a, b);
// Resultado (2) [5, 6]

```
### Acceder a los datos anidados

```javascript

const nested = [2, 4, [5, 6]];
const [a, ,[c, d]] = nested;
console.log(a, c, d);
// Resultado 2, 5, 6

```

### Asignando valores por defecto
Sino sabemos la longitud de un array, se asignan valores por defecto
```javascript

const [a, b, c] = [1, 2];
console.log(a, b, c);
// Resultado 1 2 undefined

```

Aquí asignamos los valores por defecto para no tener el valor 'undefined'

Esto puede ser útil si recibimos información de una API

```javascript

const [a=1, b=1, c=3] = [1, 2];
console.log(a, b, c);
// Resultado 1 2 3

```
